7. (There was) five people who came here last night to meet him.

➡ Opsi A yang salah secara gramatikal. Existence "there was" digunakan dalam penjabaran kata benda tunggal. Sedangkan, dalam kalimatnya, kata bendanya jamak "five people." Maka, seharusnya tertulis "there were."

8. Psychologists at Harvard University has studied the effect of the colour of a room on people's behaviour.

➡ Opsi A yang salah secara gramatikal. Auxiliary verb "has" tidak sesuai dengan bentuk subyek jamaknya "Psychologists." Ini seharusnya tertulis "have studied" untuk kesesuaian hal number antara kata kerja dengan bentuknsubyeknya.

9. Horses in the traditional society plays a very important role.

➡ Opsi C yang salah secara gramatikal. Predikat "plays" tidak menunjukkan kesesuaian hal number pada bentuk subyeknya yang jamak "Horses." Ini seharusnya tertulis "play."

10. Mia with her friend Mira always go jogging on Sunday mornings.

➡ Opsi B yang salah secara gramatikal. Subyek "Mia with her friend Mira" tetap terhitung tunggal. "With her friend Mira" hanyalah modifier dari "Mia." Ini seharusnya tertulis "always goes."

11. The girls in the living room is my classmates.

➡ Opsi C yang salah secara gramatikal. Kata kerja relasi "is" tidak menunjukkan kesesuaian hal number pada bentuk subyeknya yang jamak "The girls." Ini seharusnya tertulis "are."

12. There are some problem that we are going to discuss in the meeting.

➡ Opsi A yang salah gramatikal. Quantifier "some" menilai kuantitas kata benda. Ini seharusnya tertulis "problems" sebagai countable noun.

Adjective = kata sifat

Noun = kata benda

Relative Pronoun = kata ganti relatif

Object Pronoun = kata ganti objek

Nominal: (+) Subject + Tobe (is/am/are) + Adjective / Noun / Adverb

36. analyze of each senteses ( write down of place of word ) ​


Jawaban:
Simple Present Structure:

1) It ( Subject ) is ( Tobe ) so hot ( Complement ), with an average temperature of 34.000 centigrade ( Preposition Object ).2) It ( Subject ) suddenly ( Adverb ) expands ( Verb -s ) with a loud blast ( Preposition Object ).3) It ( Subject ) forms ( Verb -s ) a type of cloud called cumolonimbus ( Object ).4) It ( Subject ) is ( Tobe ) the reaction ( Complement ) between them ( Preposition ) which produces ( Verb -s ) static electricity ( Clause ).5) It ( Subject ) discharges ( Verb -s ) it as a lightning flash ( Preposition Object )

Penjelasan:
Analyze:

1) It ( Pronoun ) is ( Verb ) so ( Adverb ) hot ( Adjective ), with ( Preposition ) an ( Determiner ) average ( Adjective ) temperature ( Noun ) of ( Preposition ) 34.000 centigrade ( Noun ).2) It ( Pronoun ) suddenly ( Adverb ) expands ( Verb ) with ( Preposition ) a ( Determiner ) loud ( Adjective ) blast ( Noun ).3) It ( Pronoun ) forms ( Verb ) a ( Determiner ) type ( Noun ) of ( Preposition ) cloud ( Noun ) called ( Verb ) cumolonimbus ( Noun ).4) It ( Pronoun ) is ( Verb ) the ( Determiner ) reaction ( Noun ) between ( Preposition ) them ( Pronoun ) which ( Clause ) produces ( Verb ) static ( Adjective ) electricity ( Noun ).5) It ( Pronoun ) discharges ( Verb ) it ( Pronoun ) as ( Preposition ) a ( Determiner ) lightning flash ( Noun )

Pembahasan:

Simple  Present adalah bentuk yang digunakan untuk menyatakan repetisi, hal umum, atau kejadian yang terjadi di masa lampau dan terus berlanjut hingga masa sekarang - masa depan.Pola: [ Subject + Tobe is/am/are + Complement ] atau [ Subject + Verb 1 -s/-es + Object / Complement ]
