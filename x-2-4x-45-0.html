5. 1. x²-12x-45>_0 2. 4x²-33x+54>0


Pada kesempatan kali ini, soal menyajikan kita dengan 2 persamaan kuadrat. Meski dalam soal tidak diberitahukan jenis pekerjaan yang harus kita lakukan, biasanya soal semacam ini meminta kita untuk mencari akar-akar persamaan kuadrat yaitu berupa x₁ dan x₂. Selain itu, di ujung persamaan terdapat tanda ≥ serta >. Tanda ini sebagai acuan bagi kita dalam menetapkan nilai kedua akar persamaan yang akan sesuai dengan kedua persamaan kuadrat di atas.


Baiklah, silahkan ikuti pembahasan berikut untuk mengetahui nilai akar-akar persamaan dari kedua persamaan kuadrat di atas:


1. x²-12x-45 ≥ 0


a. Diskriminan (D) = b² - 4ac

D = (12)² - 4(1)(-45)

D = 144 + 180

D = 324


b. x₁ = (-b + √D)/2a

x₁ = (12 + √324)/2

x₁ = (12 + 18)/2

x₁ = 34/2

x₁ = 17


c. x₂ = (-b - √D)/2a

x₂ = (12 - 18)/2

x₂ = -6/2

x₂ = -3


Berdasarkan perhitungan ini, nilai x₁ = 17 dan x₂ = -3. Dari keduanya, hanya nilai x₁ yang memenuhi ketentuan ≥ 0. Dengan demikian, akar-akar persamaannya adalah 17.


2. 4x²-33x+54 > 0

a. Diskriminan (D) = b² - 4ac

D = (33)² - 4(4)(54)

D = 1089 + 864

D = 225


b. x₁ = (-b + √D)/2a

x₁ = (33 + √225)/8

x₁ = (33 + 15)/8

x₁ = 48/8

x₁ = 6


c. x₂ = (-b - √D)/2a

x₂ = (33 - 15)/8

x₂ = 18/8


Berdasarkan perhitungan ini, nilai x₁ = 6 dan x₂ = 18/8. Dari keduanya, nilai x₁ dan x₂ memenuhi ketentuan> 0. Dengan demikian, akar-akar persamaannya adalah 6 dan 18/8.

Maka jika x = 0, memenuhi. Jika x - 36 = 0, x = 36, memenuhi juga. Maka solusinya ada 2, x = 0 atau 36.

2. Faktorkan menjadi (x - 8)(x + 7) = 0.

x - 8 = 0, x = 8 atau x + 7 = 0, x = -7. Maka jawabannya, x = -7 atau 8.

3. Faktorkan menjadi (x + 9)(x - 5) = 0.

x + 9 = 0, x = -9 atau x - 5 = 0, x = 5. Maka jawabannya, x = -9 atau 5.

4. Faktorkan menjadi (x + 5)(x - 5) = 0.

x + 5 = 0, x = -5 atau x - 5 = 0, x = 5. Maka jawabannya, x = -5 atau 5.

Catatan tambahan:

Contoh soal nomor 1, 0 dan 36 disebut sebagai akar-akar persamaannya atau solusi dari persamaan tersebut. Akar-akar maksudnya jika dimasukkan, hasilnya akan memenuhi. Coba masukkan 1, kita akan memperoleh -35 = 0, tentu tidak memenuhi, 'kan?

Untuk soal no 2 dan 3, pemfaktoran dilakukan dengan dasar, jika a dan b akar-akar persamaan, maka

(x - a)(x - b) = 0

Jabarkan menjadi

x² - ax - bx + ab = 0

x² - (a + b)x + ab = 0

(Ingat, karena masuk ke dalam kurung dan masih minus, tandanya sama.)

Maka cari yang jumlahnya -(a + b) dan hasil kalinya ab. Ini namanya pemfaktoran.

Untuk soal no 4 kaidahnya sama dengan no 2 dan 3, namun ada pemfaktoran bentuk istimewa yang sebaiknya dihafalkan, yakni a² - b² = (a + b)(a - b). Soal seperti ini sering sekali keluar, baik pada ulangan, maupun pada olimpiade.

        [tex]\boxed{\boxed{\mathbf{(x - x_{1})(x - x_{2})=0 \to \red{kalikan \: tiap \: unsurnya}}}}[/tex]

Persamaan kuadrat juga memiliki sifat - sifat operasi terhadap akar - akar nya , sebagai berikut :

→ [tex]\sf a) \: x_{1} + x_{2} = -\frac{b}{a} \\ \sf b) \: x_{1} \times x_{2} = \frac{c}{a} \\ \sf c) \: {x_{1}}^{2} + {x_{2}}^{2} = (x_{1} + x_{2})^{2} - 2x_{1}.x_{2} \\ \sf d) \: \frac{1}{x_{1}} + \frac{1}{x_{2}} = \frac{x_{1} + x_{2}}{x_{1} \times x_{2}}[/tex]
