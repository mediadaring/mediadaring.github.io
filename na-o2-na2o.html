15. Na+O2→Na2O setarakan -oksidator-reduktor-hasil oksidasi-hasilreduksi-yang mengalami reduksi-yang mengalami oksidasi


Berdasarkan soal maka :

Reaksi setara : 2 Na + O₂ → 2 Na₂O

Maka diperoleh :  

Oksidator = mengalami reduksi = biloksnya turun = O₂

Reduktor = mengalami oksidasi = biloksnya naik = Na

Hasil Oksidasi = Na₂O

Hasil Reduksi = Na₂O

Pembahasan :  

Reaksi setara : 2 Na + O₂ → 2 Na₂O

(Reaksi Oksidasi/ Reduktor)

2 Na + O₂ → 2 Na₂O

   |__________|  

  0                    +1

Atom Na pada reaksi diatas mengalami kenaikan bilangan oksidasi dari 0 berubah menjadi +1, sehingga mengalami reaksi oksidasi atau reduktor.  

(Reaksi Reduksi/ Oksidator)

2 Na + O₂ → 2 Na₂O

            |________|  

           0                -2

Atom O pada reaksi diatas mengalami penurunan bilangan oksidasi dari 0 berubah menjadi -2, sehingga mengalami reaksi reduksi atau oksidator.  

Maka diperoleh :  

Oksidator = mengalami reduksi = biloksnya turun = O₂

Reduktor = mengalami oksidasi = biloksnya naik = Na

Hasil Oksidasi = Na₂O

Hasil Reduksi = Na₂O

Pengertian Oksidasi dan Reduksi sebagai reaksi peningkatan dan penurunan bilangan oksidasi

Oksidasi adalah reaksi dengan peningkatan bilangan oksidasi.

Zat yang mengalami kenaikan bilangan oksidasi disebut reduktor.

Contoh :

K → K⁺ + e⁻

⇒ Unsur K mengalami kenaikan bilangan oksidasi yaitu berubah dari biloks 0 menjadi +1.

Reduksi adalah reaksi dengan penurunan bilangan oksidasi.

Zat yang mengalami penurunan bilangan oksidasi disebut oksidator.

Contoh :

Cl₂ + 2e⁻ → 2 Cl⁻  

⇒ Senyawa Cl₂ mengalami penurunan bilangan oksidasi yaitu berubah dari biloks 0 menjadi -1.

29. Diketahui : 2 H2(g) + O2(g)  2 H2O() ; H = -p 4 Na(s) + O2(g)  2 Na2O(s) ; H = -q Na2O(s) + H2O()  2 NaOH(s) : H = -r Hitung : Hf . NaOH(s)


Jawaban:

Untuk menghitung entalpi pembentukan NaOH(s) (ΔHf NaOH kita dapat menggunakan persamaan termokimia yang diberikan:

2 H2(g) + O2(g) → 2 H2O(l) ; ΔH = -p

4 Na(s) + O2(g) → 2 Na2O(s) ; ΔH = -q

Na2O(s) + H2O(l) → 2 NaOH(s) ; ΔH = -r

Kita ingin mencari ΔHf NaOH(s yaitu entalpi pembentukan 1 mol NaOH(s) dari unsur-unsurnya dalam keadaan standar.

Dalam persamaan 2 Na2O(s) + 2 H2O(l) → 4 NaOH(s kita bisa lihat bahwa 2 mol NaOH(s) terbentuk dari 2 mol Na2O(s) dan 2 mol H2O(l).

Mari kita ubah persamaan ketiga agar Na2O(s) dan H2O(l) berada dalam jumlah yang sesuai:

Na2O(s) + 1/2 H2O(l) → NaOH(s) ; ΔH = -(r/2)

Sekarang kita bisa bandingkan persamaan ketiga dengan persamaan pertama untuk menghilangkan Na2O(s). Kita perlu menggandakan persamaan ketiga dan mengalikan persamaan pertama dengan angka yang sesuai:

2(Na2O(s) + 1/2 H2O(l)) → 2NaOH(s) ; ΔH = -r

4 H2(g) + 2 O2(g) → 4 H2O(l) ; ΔH = -2p

Kemudian kita tambahkan kedua persamaan tersebut:

2(Na2O(s) + 1/2 H2O(l)) + 4 H2(g) + 2 O2(g) → 2NaOH(s) + 4 H2O(l) ; ΔH = -r -2p

Karena kita ingin mencari entalpi pembentukan NaOH(s yang artinya reaksi ini melibatkan satu mol NaOH(s kita dapat menyederhanakan persamaan tersebut:

2(Na2O(s) + 1/2 H2O(l)) + 4 H2(g) + 2 O2(g) → 2NaOH(s) + 4 H2O(l) ; ΔH = -r -2p

Dalam persamaan di atas Na2O(s) + 1/2 H2O(l) adalah bagian dari NaOH(s) dalam entalpi pembentukan. Jadi ΔHf NaOH(s) = -r/2 - 2p.
