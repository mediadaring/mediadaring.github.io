23. Complete with the correct form of the verb be


Maria : Hello, Nancy. This is Marta and this is Carlos.

Nancy : Hi. Are you from Mexico?

Carlos : No, we aren't. I am from Spain and she is from Peru.

Pembahasan

Verb be adalah kata kerja bantu yang digunakan untuk kalimat nominal (tidak memiliki verb).

Be dapat berupa is, am and are.

Berikut subjek dan verb be yang digunakan, yaitu :

I => amYou => areShe => isHe => isWe => areThey => areIt => is

★ Simple Present Tense

adalah bentuk tense  yang digunakan untuk menyatakan fakta, kebenaran umum, kebiasaan/kegiatan rutin, dan situasi permanen

★ Ket. Waktu Simple Present Tense

Every-day/week/month/year

Once/twice-a day

Always

Usually

Often

★ Rumus Simple Present Tense

–Verbal–

Subject : I, You, They, We

(+) S + V1

(–) S + Do + not + V1

(?) Do + S + V1

Subject : She, He, It

(+) S + V1 (-s/es)

(–) S + Does + not + V1

(?) Does + S + V1

–Nominal–

(+) S + To Be + Noun/Adj/Adv

(–) S + To Be + not + Noun/Adj/Adv

(?) To Be + S + Noun/Adj/Adv

→ To be : am, are, is
