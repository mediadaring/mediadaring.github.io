19. Esktensi yang dihasilkan oleh file spreadsheet atau ms. excel adalah?


Jawaban:

xlsx atau xslm

Penjelasan:

Ekstensi (atau ekstensi file) yang dihasilkan oleh file spreadsheet atau Microsoft Excel adalah .xlsx (atau .xlsm untuk file Excel yang mengandung makro). Ekstensi .xlsx adalah ekstensi default untuk file Excel yang dibuat di versi terbaru program tersebut, seperti Excel 2013, Excel 2016, Excel 2019, dan Excel untuk Microsoft 365. Ekstensi ini merupakan singkatan dari "XML Spreadsheet" dan merupakan format file baru yang diperkenalkan oleh Microsoft Office 2007, menggantikan format file Excel sebelumnya yang berakhiran .xls. Format file .xlsx menggunakan kompresi ZIP untuk mengurangi ukuran file, memungkinkan penggunaan lebih banyak fitur dan kapasitas penyimpanan yang lebih besar daripada format file sebelumnya.

31. Format File MS Excel 2010 Adalah​


Jawaban:

xlsx

Penjelasan:

Dalam Ms. Excel sebelumnya (versi 97, 200 dan 2003) format filenya adalah xls. Namun pada Ms. Excel versi berikutnya (versi 2007, 2010 dan seterusnya), format filenya menjadi xlsx.

Maksudnya adalah, bahwa file tersebut didukung oleh xml yang mampu meringkas dan memadatkan kapasitas penyimpanan file.
