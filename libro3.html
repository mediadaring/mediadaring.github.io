2. bilangan oksidasi masing masing unsur dalam senyawa atau ion dari LiBrO3


Suatu senyawa LiBrO3 memiliki nilai bilangan oksidasi masing-masing unsur penyusun : Li=+7, Br=-1; O=-2 sesuai aturan bahwa senyawa memiliki total biloks sama dengan nol

Li + Br + 3(O) = 0

+7 -1 + 3(-2) = 0

0 = 0

(Terbukti)

Pembahasan

Reaksi redoks merupakan reaksi yang saling berkesinambungan. Dimana terjadi oksidasi maka akan terjadi oksidasi pula. Reaksi oksidasi merupakan reaski pelepasan elektron, molekul atau ion. reaksi ini juga ditandai dengan peningkatan bilangan oksisadi. Sedangkan reaksi reduksi merupakan reaksi penangkapan elektron, molekul atau ion. dengan kata lain, dalam reaksi reduksi terjadi pengurangan bilangan oksidasi.

Pembahasan

Penentuan bilangan oksidasi suatu unsur mengikuti aturan aturan sebagai berikut :

Bilangan oksidasi unsur bebas adalah nol. Unsur bebas misalnya Na, H₂, Au, Fe, dsb).

        Biloks Br dalam Br₂ = 0

        Biloks Zn                  = 0

        Biloks Ag                 = 0

Bilangan oksidasi ion monoatom atau ion sederhana adalah sama dengan muatannya. Ion sederhanya misalnya Na⁺, Ca²⁺, Cl⁻ dsb.

        Biloks Al³⁺                = +3

        Biloks Zn²⁺               = +2

        Biloks F⁻                   = -1

        Biloks H dalam HBr          = +1

        Biloks H dalam H₂O         = +1

        Biloks H dalam KH           = -1

        Biloks H dalam BaH₂       = -1

Bilangan oksidasi unsur oksigen dalam senyawa atau ion polyatom adalah -2, kecuali dalam senyawa peroksida (Na₂O₂, H₂O₂) mempunyai biloks -1 dan dalam senyawa OF₂ adalah +2.

        Biloks O dalam H₂O                   = -2

        Biloks O dalam CO₃²⁻                = -2

        Biloks O dalam K₂O₂                  = -1

Bilangan oksidasi unsur - unsur golongan I A dalam senyawa sama dengan +1, unsur - unsur golongan II A  sama dengan +2, dan unsur golongan III A dalam senyawa sama dengan +3.

Tentukan bilangan oksidasi unsur dalam senyawa berikut! C. d. CaH₂ ZnF 2​


Penjelasan:

Untuk menentukan bilangan oksidasi unsur dalam senyawa yang diberikan, kita perlu mengingat beberapa aturan:

1. Bilangan oksidasi unsur dalam keadaan bebas (unsur murni) adalah 0.

2. Bilangan oksidasi hidrogen (H) dalam sebagian besar senyawa adalah +1, kecuali saat terikat dengan logam dengan bilangan oksidasi negatif (biasanya logam alkali dan logam alkali tanah) seperti LiH, NaH, CsH, dll., di mana bilangan oksidasi H adalah -1.

3. Bilangan oksidasi oksigen (O) dalam sebagian besar senyawa adalah -2, kecuali dalam peroksida (seperti H₂O₂) di mana bilangan oksidasi O adalah -1.

4. Bilangan oksidasi logam alkali (seperti Li, Na, K) adalah +1, dan bilangan oksidasi logam alkali tanah (seperti Mg, Ca, Ba) adalah +2.

Dengan menggunakan aturan-aturan tersebut, kita dapat menentukan bilangan oksidasi unsur dalam senyawa yang diberikan:

a. SbOCl₅

Bilangan oksidasi Sb = x (asumsi)

Bilangan oksidasi O = -2 (aturan)

Bilangan oksidasi Cl = -1 (aturan)

Karena total bilangan oksidasi dalam senyawa harus nol, maka kita dapat menghitung:

x + 5(-1) + (-2) = 0

x - 5 - 2 = 0

x = +7

Jadi, bilangan oksidasi Sb dalam senyawa SbOCl₅ adalah +7.

b. LiBrO₃

Bilangan oksidasi Li = +1 (aturan)

Bilangan oksidasi Br = x (asumsi)

Bilangan oksidasi O = -2 (aturan)

Karena total bilangan oksidasi dalam senyawa harus nol, maka kita dapat menghitung:

(+1) + x + 3(-2) = 0

1 + x - 6 = 0

x - 5 = 0

x = +5

Jadi, bilangan oksidasi Br dalam senyawa LiBrO₃ adalah +5.

c. CaH₂

Bilangan oksidasi Ca = +2 (aturan)

Bilangan oksidasi H = x (asumsi)

Karena total bilangan oksidasi dalam senyawa harus nol, maka kita dapat menghitung:

(+2) + 2x = 0

2 + 2x = 0

2x = -2

x = -1

Jadi, bilangan oksidasi H dalam senyawa CaH₂ adalah -1.

d. ZnF₂

Bilangan oksidasi Zn = x (asumsi)

Bilangan oksidasi F = -1 (aturan)

Karena total bilangan oksidasi dalam senyawa harus nol, maka kita dapat menghitung:

x + 2(-1) = 0

x - 2 = 0

x = +2

Jadi, bilangan oksidasi Zn dalam senyawa ZnF₂
