5. Kak tolongin Ngerjain MTK buat belajar untuk try out besok​


Jawaban:

B. 864 cm^2

Penjelasan dg langkah2:

lebar karton = 40 cm

pajang karton = 60 cm

lebar foto = 40 - 4 - 4 = 32 cm

foto dan karton sebangun, maka

pjg foto / pjg karton = lebar foto / lebar karton

pjg foto / 60 cm = 32 cm / 40 cm

pjg foto = (60 cm x 32 cm) / 40 cm

pjg foto = 48 cm

L karton = 40 cm x 60 cm = 2.400 cm^2

L foto = 32 cm x 48 cm = 1.536 cm^2

L karton yg tdk tertutup foto:

L = L karton - L foto

L = 2.400 cm^2 - 1.536 cm^2

= 864 cm^2

12. #Try out UTBK#Kerjakan menggunakan cara ​


Jawaban:

Pilihan yang sesuai adalah  ( D )  Pernyataan  ( 1 )  SAJA cukup untuk menjawab pertanyaan, dan pernyataan  ( 2 )  SAJA cukup.

Penjelasan:

Ditanya:

Berapa nilai x

Pernyataan:

(1) 7AB = 5BC = 5CA

(2) 6AB = 5BC = 4CA

Jawab:

Untuk pernyataan (1)

7AB = 5BC

BC = [tex]\frac{7}{5} AB[/tex]

7AB = 5CA

CA  = [tex]\frac{7}{5} AB[/tex]

Dengan menggunakan aturan cosinus.

BC² = AB² + AC² - 2AB.AC.cos x

cos x = [tex]\frac{AB^2+AC^2-BC^2}{2.AB.AC}[/tex]

        = [tex]\frac{AB^2+(\frac{7}{5}AB)^2-(\frac{7}{5}AC)^2 }{2.AB.\frac{7}{5}AB}[/tex]

        = [tex]\frac{AB^2+\frac{49}{25}AB^2-\frac{49}{25}AB^2 }{\frac{14}{5}AB^2}[/tex]

        = [tex]\frac{AB^2}{\frac{14}{5}AB^2}[/tex]

        = [tex]\frac{1}{\frac{14}{5} }[/tex]

        = [tex]\frac{5}{14}[/tex]

x = arc cos [tex]\frac{5}{14}[/tex]

Untuk pernyataan (2) caranya sama, dengan menggunakan aturan cosinus.

Dengan demikian

Pilihan yang sesuai adalah  ( D )  Pernyataan  ( 1 )  SAJA cukup untuk menjawab pertanyaan, dan pernyataan  ( 2 )  SAJA cukup.
