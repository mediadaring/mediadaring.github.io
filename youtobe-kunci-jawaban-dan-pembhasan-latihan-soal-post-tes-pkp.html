19. Tolong pembhasan nya kak #terimakasih


Jawab:

Jumlah uang yang harus dibayar adalah Rp. 40.000

Penjelasan dengan langkah-langkah:

1 lusin sendok = s

1 lusin piring = p

Harga 3 lusin sendok dan 4 lusin piring adalah Rp. 145.000.

3s + 4p = 145.0000 ..... (i)

Harga 2 lusin sendok dan 3 lusin piring adalah Rp. 105.000

2s + 3p = 105.000 ...... (ii)

3s + 4p = 145.000 ..... × 3

2s + 3p = 105.000 ..... × 4

Menjadi :

9s + 12p = 435.000

8s + 12p = 420.000 _

s = 15.000

disubtitusi ke salah satu persamaan :

2s + 3p = 105.000

2. (15.000) + 3p = 105.000

30.000 + 3p = 105.000

3p = 105.000 - 30.000

3p = 75.000

p = 75.000 ÷ 3

p = 25.000

Harga 1 lusin sendok dan 1 lusin piring

= s + p

= 15.000 + 25.000

= Rp. 40.000

Sendok = y

Piring = x

( 3y + 4x = 145.000 ) 6y + 8x = 290.000

( 2y + 3x = 105.000 ) × 6y + 9x = 315.000 - x = 25.000

X = 25.000

2y + 3 × 25.000 = 105.000

2y + 75.000 = 105.000

2y + 105.000 - 75.000 = 30.000

2y = 30.000

y = 15.000

Jadi 1 lusin sendok = 25.000

1 lusin piring piring = 15.000

25.000 + 15.000 = Rp 40.000

Maka jumlah yang harus dibayar Rp 40.000
