9. Pengukuran suhu ruangan menunjukkan 310 k maka skala °c, °f, dan °r, menunjukkan


310 k = 5/5 x (310 - 273)° C
= 1 x 37 = 37° c

310k = 9/5 x (310-273) + 32
= 9/5 x 37 + 32
= 98,6° f


310 k = 4/5 x (310 - 273)
= 4/5 x 37
= 29,6 ° rMata Pelajaran = Fisika
Materi = Konversi suhu C,F,R, K


Jika diketahui suhu pada kelvin, untuk mengubahnya ke satuan lain lebih mudahnya ubah dahulu ke celcius dengan menguranginya dengan 273 (ini sudah ketentuannya)

==Konversi Kelvin ke Celcius==

310 °K = 310-273 = 37 °C

Perbandingan Celcius : Reamur : Fahrenheit yaitu 5 : 4 : 9.
Tetapi, jika mengubah dari satuan lain ke satuan Fahrenheit, setelah mendapat hasil dari perbandingan, harus ditambah 32. Jika dari Fahrenheit ke satuan lain, sebelum dihitung menggunakan perbandingan, angkanya terlebih dahulu dikurang 32.

Untuk mempermudah dalam mengubah ke Fahrenheit, kita gunakan saja skala Celcius.

==Konversi Celcius ke Fahrenheit==

37 °C = 9/5 × 37 = 66,6 + 32 = 98,6 °F

Dalam mengubah ke Reamur, kita gunakan juga skala Celcius agar lebih mudah.

==Konversi Celcius ke Reamur==

37 °C = 4/5 × 37 = 29,6 °R

21. Pengukuran suhu ruangan menunjukkan 310 k maka skala Â°c, Â°f, dan Â°r, menunjukkan


0°C = 273K
jadi, °C = 310-273
= 37°C
untuk skala F gunakan perbandingan terhadap C yaitu, 9/5 x (skala dalam C) +32
9/5 x 37 +32
= 98.6°F
untuk R, yaitu dengan perbandingan dengan C, bisa juga dengan F, R:C = 4:5
4/5 x 37
= 29,6°RMapel : Fisika
Materi : Suhu
Kelas : Vll
Kata Kunci : konversi suhu
Kode Kategori : 7.6.3
KTSP
Pembahasan : 

Diketahui suhu yang diukur termometer Kelvin adalah 310°K. Pertanyaannya, berapa jika diukur dengan termometer Fahrenheit, Celcius dan Reamur.

Kelvin ke Celcius
(K - 273)/100 = C/100
K - 273 = C
310 - 273 = C
C = 37°C. ←

Kelvin ke Fahrenheit
(F - 32)/180 = (K - 273)/100
(F - 32)/180 = (310 - 273)/100
(F - 32)/180 = 37/100
F - 32 = 37/100 × 180
F - 32 = 66,6
F = 66,6 + 32
F = 98,6°F. ←

Kelvin ke Reamur
(K - 273)/100 = R/80
(310 - 273)/100 = R/80
37/100 = R/80
37/100 × 80 = R
29,6 = R
R = 29,6°R. ←
