15. jawaban soal sbmptn 2016 tkd saintek kode 201dalam suasana basa,cl2 mengalami reaksi disproporsionasi menghasilkan ion cl- dan clo3-. jumlah mol ion clo3- yang dihasilkan daei 1 mol cl2 adalah...


nih saya bantuin:
persamaan reaksi setaranya :
3Cl₂ + 6 OH⁻ ---->5 Cl⁻ + ClO₃⁻ + 3 H₂O

perbandingan mol = perbandingan koefisien
mol ClO₃⁻ / mol Cl₂ = koef ClO₃⁻  / koef Cl₂  lalu kali silangkan

mol ClO₃⁻ =(koef ClO₃⁻ / koef Cl₂) . mol Cl₂

mol ClO₃⁻  = (1/3) . 1 mol = 1/3 mol

jadi jawabannya dari 1 mol Cl₂ bisa menghasilkan 1/3 mol ClO₃⁻

Jawaban:

D. 80

Penjelasan dengan langkah-langkah:

jarak = 500 km

kecepatan = x km/jam

waktu = w + 3/4

jarak = kecepatan x waktu

500 = x (w + 3/4)

500 = xw + 3/4x ........ persamaan I

jarak = 400

kecepatan = x km/jam

w = w - 1/2

400 = x (w-1/2)

400 = xw -1/2x .............. persamaan II

persamaan I - Persamaan II (metode eliminasi xw)

100 = 3/4x - (-1/2x)

100 = 3/4x + 1/2x

100 = (3+2) /4 x

100 = 5/4 x

x = 100 x 4/5

x = 80 km/jam (D)

s1 = 500 km

w1 = t + 3/4 jam

s2 = 400 km

w2 = t - 1/2 jam

s1/w1 = s2/w2

500/(t + 3/4) = 400/(t - 1/2)

500(t - 1/2) = 400(t + 3/4)

500t - 250 = 400t + 300

500t - 400t = 300 + 250

100t = 550

t = 550/100

t = 5,5 jam

v1 = s1/w1

v1 = s1/(t + 3/4)

v1 = 500/(5,5 + 3/4)

v1 = 500/(5,5 + 0,75)

v1 = 500/6,25

v1 = 80

jadi x adalah 80 km/jam (D)
