24. Pada gawai yang kamu miliki terdapat dua browser yaitu opera dan chrome. Pengaturan default browser pada gawai tersebut adalah opera. Karena kamu ingin mengganti setting browser menjadi chrome, maka jelaskanlah secara ringkas langkah-langkah untuk mengatur setting default browser pada gawai yang kamu miliki ?


Jawaban:

1. Buka aplikasi Google Chrome pada gawai

2. Klik customize and control Google Chrome dipojok kanan atas

3. Pilih Settings

4. Scroll atau pilih bagian Default Browser

5. Klik Make Default

Penjelasan:

Langkah-langkah yang dapat dilakukan untuk membuat aplikasi browser menjadi default pada gawai:

1.  Buka aplikasi web browser yang ingin dijadikan sebagai default gawai, biasanya web browser default saat kondisi gawai masih baru antara Internet Explorer atau Opera, pengguna atau pemilik gawai bisa mendownload web browser lain yang diinginkan menggunakan web browser yang sudah ada terlebih dahulu. Langkah-langkah ini akan dicontohkan menggunakan aplikasi web browser Google Chrome.

2. Buka customize and control Google Chrome dipojok kanan atas (titik 3), Pilih Settings

3. Pilih opsi default browser, klik make default. Secara otomatis Chrome akan menjadi default web browser pada gawai yang digunakan.

Jika ingin mengembalikan aplikasi web browser default ke Opera, menggunakan cara yang hampir sama:

1.   Buka aplikasi web browser Opera

2. Pilih Menu pada bagian pojok kiri atas kemudian pilih setting (Alt+P)

3. Scroll ke bawah atau gunakan search engine di kanan atas untuk menemukan Default Browser

4. Klik Make Default, secara otomatis, default browser yang sebelumnya Google Chrome akan tergantikan oleh Opera

29. Jelaskan cara kerja web browser


Sebelum kita mengakses web, client atau pengguna harus bias mengoperasikan web browser sebagai alat untuk bias masuk ke dunia maya.
Client/user mengetikkan sebuah alamat web didalam browser yang dia gunakan. Setelah menekan tombol ernter atau mengklik tanda search pada web browser yang digunakan, web browser tadi mengirim sebuah permintaan atau request ke web server melalui protokol-protokol yang diperlukan dalam request ke web server seperti protocol HTTP request.
Setelah itu menerima permintaan dan mencari halaman web atau web file yang ada di dalam server seperti yang direquest. Setelah didapat, server mengirimkannya kembali web file yang diminta ke user melalui prtokol HTTP respone ke web browser yang dugunakan user. Selanjutnya user bisa memakai web file yang diterimanya.
