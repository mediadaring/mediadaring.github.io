27. Persamaan kuadrat 2x² - 2x + 6 = 0 akar-akarnya x₁ dan x₂ , maka persamaan baru yang akar-akarnya 3x₁ dan 3x₂ adalah ........ A.  x² - 3x +27 = 0 B. x² - 3x - 27 = 0 C. x² + 3x + 27 = 0 D. x² + 3x - 9 = 0 E. x² + 3x + 9 = 0tolong gimana nih penyelesaiannya ...


3x₁ ⇒ α ; dan 3x₂ ⇒β
2x² - 2x + 6 = 0
Maka :
a = 2 ; b = - 2 ; c = 6
Penyelesaian :
⇒ x₁ + x₂ = -b/a = -(-2)/2 = 1
⇒ x₁ · x₂ = c/a = 6/2 = 3
Sehingga :
α + β = 3x₁ + 3x₂
         = 3 (x₁ + x₂)
         = 3 (1)
         = 3

α·β = (3x₁) · (3x₂)
      = 9x₁.x₂
      = 9(3)
      = 27

Jadi, Persamaan Kuadrat Baru :
Rumus : x² - (α+β)x + α·β = 0
⇒ x² - 3x + 27 = 0     (A)
