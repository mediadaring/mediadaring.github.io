16. 1) Tentukan nilai dari 3 a. log 45+ log 60+log 14-³ log 20-³log 70 b. 4. log 18+2. log 6-3. 'log 12 (2) Tentukan himpunan penyelesaian dari persamaan berikut a. 2log (3x-7)=³log (3x-7) b. 2log (2x²-7x-30)=2x-3)log (x+12) c. log²x + log x-log 25=0​


1) a. Untuk menyelesaikan masalah ini, kita dapat menggunakan sifat-sifat logaritma sebagai berikut:

log a + log b = log ab

log a - log b = log (a/b)

n log a = log aⁿ

Menerapkan sifat-sifat tersebut pada rumus yang diberikan, kita dapat menyederhanakan menjadi:

log 45 + log 60 + log 14 - ³log 20 - ³log 70

= log (45 x 60 x 14) - log (20³) - log (70³)

= log (37,800) - log (8,000) - log (343,000)

= log [(37,800 / 8,000) / 343,000]

= log (0.00125)

Jadi, nilai dari 3a adalah -2.9031.

b. Dalam masalah ini, kita juga dapat menggunakan sifat-sifat logaritma untuk menyederhanakan rumus:

4log18 + 2log6 - 3log12

= log18⁴ + log6² - log12³

= log (104,976) + log (36) - log (1,728)

= log [(104,976 x 36) / 1,728]

= log (2,196)

Jadi, nilai dari 3b adalah 3.3410.

2) a. Persamaan yang diberikan adalah:

2log(3x-7) = ³log(3x-7)

Kita dapat menyelesaikan persamaan ini dengan mengubah bentuk kubik menjadi bentuk pangkat tiga biasa:

2log(3x-7) = log[(3x-7)³]

log[(3x-7)²] = 0

(3x-7)² = 1

3x-7 = ±1

x = 2 atau x = 4

Maka, himpunan penyelesaian dari persamaan a adalah {2, 4}.

b. Persamaan yang diberikan adalah:

2log(2x²-7x-30) = (2x-3)log(x+12)

Kita dapat menyelesaikan persamaan ini dengan mengubah bentuk kubik menjadi bentuk pangkat tiga biasa:

log[(2x²-7x-30)²] = log[(x+12)^(2x-3)]

(2x²-7x-30)² = (x+12)^(2x-3)

Namun, persamaan ini tidak dapat diselesaikan secara eksplisit. Maka, kita dapat menggunakan metode numerik untuk mencari akar persamaan ini.

c. Persamaan yang diberikan adalah:

log²x + log x - log 25 = 0

Kita dapat menyederhanakan persamaan ini dengan menggabungkan kedua suku pertama menggunakan sifat logaritma:

log x (log x + 1) - log 25 = 0

log x (log x + 1) = log 25

log x² + log x - log 25 = 0

log [(x²)(x)] - log 25 = 0

log (x³ / 25) = 0

x³ / 25 = 1

x³ = 25

x = ∛25

Maka, himpunan penyelesaian dari persamaan c adalah {∛25}.

ᵃlog a = 1

ᵃlog 1 = ᵃlog a⁰ = 0

ᵃlog mn = ᵃlog m + ᵃlog n

ᵃlog m/n = ᵃlog m - ᵃlog n
