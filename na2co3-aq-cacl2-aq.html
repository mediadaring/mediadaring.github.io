10. setrakan persamaan reaksi Na2CO3 (s)+HCl (aq)-> NaCl (aq)+H2O (l)+CO2 (g)


Penyetaraan reaksi berikut : Na₂CO₃ (s) + HCl (aq) ⇒ NaCl (aq) + CO₂ (g) + H₂O (l) yaitu:

Langkah 1 : Berdasarkan reaksi, hitung jumlah atom kiri dan kanan dengan menggunakan cara sebagai berikut :

Reaksi kiri :                             Reaksi kanan :

Atom Na = 2                             Atom Na = 1

Atom C = 1                                Atom C = 1

Atom H = 1                                Atom H = 2

Atom O = 3                               Atom O = 3

Atom Cl = 1                               Atom Cl = 1

Langkah 2 : Setarakan atom Na pada reaksi kanan dengan menambahkan koefisien 2.

Na₂CO₃ (s) + HCl (aq) ⇒ 2NaCl (aq) + CO₂ (g) + H₂O (l)

Langkah 3 : Setarakan atom Cl pada reaksi kiri dengan menambahkan koefisien 2.

Na₂CO₃ (s) + 2HCl (aq) ⇒ 2NaCl (aq) + CO₂ (g) + H₂O (l)

Berdasrkan penyetaraan diatas, maka setiap unsur di reaksi kiri dan kanan sudah setara.

∴ Kesimpulan reaksi setaranya : Na₂CO₃ (s) + 2HCl (aq) ⇒ 2NaCl (aq) + CO₂ (g) + H₂O (l).

Ca ⇒ a = c
Cl ⇒ 2a = d
Na ⇒ 2b = d
S ⇒ b = c
O ⇒ 4b = 4c ⇒ b = c

Jika a = 1, maka c = 1, b = 1, dan d = 2

Reaksi setara ⇒ CaCl₂ + Na₂SO₄ → CaSO₄ + 2NaCl
