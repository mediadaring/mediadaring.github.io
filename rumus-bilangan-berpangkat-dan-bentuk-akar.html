JAWABANNYA:

Penjelasan dengan langkah-langkah:

1. Suatu perpangkatan bila basisnya sama maka berlaku seperti berikut:

[tex] {a}^{x} \times {a}^{y} = {a}^{x + y} \\ \frac{ {a}^{x} }{ {a}^{y} } = {a}^{x - y} [/tex]

Maka,

[tex] = {5}^{1} \times {5}^{3} \times {5}^{ - 2} \\ = {5}^{1 + 3 - 2} \\ = {5}^{2} [/tex]

2. Pertama-tama ada cara yang mudah seperti pemfaktoran ( selisih kuadrat )berikut:

[tex] {a}^{2} - {b}^{2} = (a + b)(a - b)[/tex]

Maka, tinggal memasukkan angka pada soal dengan rumus tersebut:

[tex] = ( \sqrt{8 } + \sqrt{3} )( \sqrt{8} - \sqrt{3} ) \\ = { \sqrt{8} }^{2} - { \sqrt{3} }^{2} \\ = 8 - 3 \\ = 5 [/tex]

Namun dapat dijabarkan seperti berikut:

[tex](a + b)(a - b) = {a}^{2} - ab + ab - {b}^{2} [/tex]

Masukkan bilangan pada soal

[tex] = ( \sqrt{8} + \sqrt{3} )( \sqrt{8} - \sqrt{3} ) \\ = { \sqrt{8} }^{2} - \sqrt{24} + \sqrt{24} - { \sqrt{3} }^{2} \\ = 8 - 3 \\ = 5[/tex]

Oh iya, mengapa

[tex] { \sqrt{8} }^{2} = 8[/tex]

Karena,

[tex] { \sqrt{a} }^{2} = {a}^{2 \times \frac{1}{2} } = a\\ { \sqrt{8} }^{2} = {8}^{2 \times \frac{1}{2} } = 8[/tex]
