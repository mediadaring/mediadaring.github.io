2. Apakah perbedaan Reformasi dalam kementerian sosial dengan kementerian-kementrian sosial sebelumnya? ​.


Reformasi dalam kementerian sosial adalah upaya untuk meningkatkan efisiensi dan efektivitas dalam pelayanan sosial. Ini termasuk meningkatkan kualitas pelayanan, mengurangi biaya, meningkatkan aksesibilitas, dan meningkatkan keterlibatan masyarakat. Reformasi juga melibatkan peningkatan transparansi dan akuntabilitas, serta peningkatan keterlibatan pemerintah daerah dan swasta dalam pelayanan sosial. Perbedaan utama antara reformasi dalam kementerian sosial dengan kementerian-kementrian sosial sebelumnya adalah bahwa reformasi lebih berfokus pada peningkatan efisiensi dan efektivitas pelayanan sosial, serta peningkatan transparansi dan akuntabilitas. Reformasi juga mencakup peningkatan keterlibatan masyarakat dan pemerintah daerah dalam pelayanan sosial.

Penjelasan dengan langkah-langkah:

jarak = 500 km

kecepatan = x km/jam

waktu = w + 3/4

jarak = kecepatan x waktu

500 = x (w + 3/4)

500 = xw + 3/4x ........ persamaan I

jarak = 400

kecepatan = x km/jam

w = w - 1/2

400 = x (w-1/2)

400 = xw -1/2x .............. persamaan II

persamaan I - Persamaan II (metode eliminasi xw)

100 = 3/4x - (-1/2x)

100 = 3/4x + 1/2x

100 = (3+2) /4 x

100 = 5/4 x

x = 100 x 4/5

x = 80 km/jam (D)

s1 = 500 km

w1 = t + 3/4 jam

s2 = 400 km

w2 = t - 1/2 jam

s1/w1 = s2/w2

500/(t + 3/4) = 400/(t - 1/2)

500(t - 1/2) = 400(t + 3/4)

500t - 250 = 400t + 300

500t - 400t = 300 + 250

100t = 550

t = 550/100

t = 5,5 jam

v1 = s1/w1

v1 = s1/(t + 3/4)

v1 = 500/(5,5 + 3/4)

v1 = 500/(5,5 + 0,75)

v1 = 500/6,25

v1 = 80

jadi x adalah 80 km/jam (D)
