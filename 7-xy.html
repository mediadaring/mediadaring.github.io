19. jika ada persamaan x^2 - xy + y^2 = 7 dan x - xy + y= -1 maka nilai x+y adalah


x - xy + y = -1
• x + y = xy - 1

x² - xy + y² = 7
• x² + y² = xy + 7

#ingat :
(x + y)² = (x + y)(x + y)
(x + y)² = x² + 2xy + y²
(x + y)² - 2xy = x² + y²

maka,
x² + y² = xy + 7
(x + y)² - 2xy = xy + 7
(xy - 1)² - 2xy = xy + 7

agar mudah dipahami, anggap xy mwrupakan satu variabel, misal xy = a

(a - 1)² - 2a = a + 7
a² - 2a + 1 - 2a - a - 7 = 0
a² - 5a - 6 = 0
(a - 6)(a + 1) = 0
a = 6 atau a = -1

• untuk xy = 6
x - xy + y = -1
x - 6 + y = -1
x + y = 6 - 1
x + y = 5

• untuk xy = -1
x - xy + y = -1
x - (-1) + y = -1
x + 1 + y = -1
x + y = -1 - 1
x + y = -2
