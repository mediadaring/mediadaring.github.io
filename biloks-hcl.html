12. biloks dari 2nah2po3 ?biloks dari 2ph3 ?​


Jawaban:

Biloks dari 2NaH2PO3 (natrium hipofosfit) adalah:

- Biloks natrium (Na) adalah +1 karena ia adalah logam alkali.

- Biloks hidrogen (H) adalah +1 karena ia umumnya memiliki bilangan oksidasi positif ketika terikat dengan nonlogam.

- Biloks fosfor (P) adalah +1 karena ada dua hidrogen yang terikat ke fosfor dan masing-masing hidrogen memiliki bilangan oksidasi +1. Namun, ada dua atom oksigen di sebelah fosfor, yang masing-masing memiliki bilangan oksidasi -2. Karena total muatan dari senyawa harus netral, maka bilangan oksidasi dari fosfor harus +1 untuk menetralkan total bilangan oksidasi hidrogen dan oksigen. Oleh karena itu, bilangan oksidasi fosfor dalam 2NaH2PO3 adalah +1.

Jadi, biloks dari 2NaH2PO3 adalah: Na(+1) + 2H(+1) + P(+1) + 3O(-2) = 0.

Sedangkan, biloks dari 2PH3 (difosfina) adalah:

Biloks fosfor (P) adalah -3 karena ia terikat dengan tiga atom hidrogen (H), dan masing-masing hidrogen memiliki bilangan oksidasi +1. Oleh karena itu, untuk menetralkan total bilangan oksidasi hidrogen dan fosfor harus -3.

Jadi, biloks dari 2PH3 adalah: 2P(-3) + 6H(+1) = 0.

14. 1. Biloks Mn pada K2MnO4 2. Biloks N pada NH4+ 3. Biloks Cr pada K2CrO7 4. Biloks O pada Na2O2 5. BILOKS Mn pada KMnO4 6. Biloks S pada Na2SO4 7. Biloks H pada Na2H2


1. Biloks Mn pada K₂MnO₄ adalah +6.

2. Biloks N pada NH₄⁺ adalah -3.

3. Biloks Cr pada K₂CrO₇ adalah +12.

4. Biloks O pada Na₂O₂ adalah -1.

5. Biloks Mn pada KMnO₄ adalah +7.

6. Biloks S pada Na₂SO₄ adalah +6.

7. Biloks H pada Na₂H₂ adalah -1.

Pembahasan : 1. Biloks Mn pada K₂MnO₄

biloks K = +1 (merupakan unsur golongan IA)  

aturan biloks :  

Bilangan oksidasi unsur golongan utama (IA,IIA,IIIA,IVA,VA,VIA) sesuai dengan golongannya. Contoh : IA = +1, IIA = +2, IIIA = +3, IVA = -4, VA = -3 , VIA = -2 , VIIA = -1 , VIIIA = 0.  

biloks O = -2

aturan biloks :  

Bilangan oksidasi dalam senyawa O adalah -2 kecuali dalam peroksida (-1) dan dalam senyawa biner dengan fluor (+2). Contoh : Bilangan oksidasi O dalam H₂O = -2 , kecuali bilangan oksidasi O dalam H₂O₂ dan BaO₂ = -1 , dan bilangan oksidasi O dalam OF = +2.

biloks K₂MnO₄ = 0  

(2 × biloks K) + biloks Mn + (4 × biloks O) = 0  

(2 × (+1)) + biloks Mn + (4 × (-2)) = 0  

(+2) + biloks Mn + (-8) = 0  

biloks Mn + (-6) = 0  

biloks Mn = +6

2. Biloks N pada NH₄⁺

Biloks H = +1

aturan biloks :  

Bilangan oksidasi hidrogen dalam senyawanya adalah +1 kecuali dalam Hidrida, atom Hidrogen biloksnya adalah -1. Contoh : Bilangan oksidasi H dalam H₂O, NH₃, dan HCl = +1, kecuali Bilangan oksidasi H dalam NaH dan CaH₂ = -1.

biloks NH₄⁺ = +1

biloks N + (4 × biloks H) = +1

biloks N + (4 × (+1)) = +1

biloks N + (+4) = +1

biloks N = (+1) - 4

biloks N = -3

3. Biloks Cr pada K₂CrO₇

biloks K = +1 (merupakan unsur golongan IA)  

aturan biloks :  

Bilangan oksidasi unsur golongan utama (IA,IIA,IIIA,IVA,VA,VIA) sesuai dengan golongannya. Contoh : IA = +1, IIA = +2, IIIA = +3, IVA = -4, VA = -3 , VIA = -2 , VIIA = -1 , VIIIA = 0.  

biloks O = -2

aturan biloks :  

Bilangan oksidasi dalam senyawa O adalah -2 kecuali dalam peroksida (-1) dan dalam senyawa biner dengan fluor (+2). Contoh : Bilangan oksidasi O dalam H₂O = -2 , kecuali bilangan oksidasi O dalam H₂O₂ dan BaO₂ = -1 , dan bilangan oksidasi O dalam OF = +2.

biloks K₂CrO₇ = 0  

(2 × biloks K) + biloks Cr + (7 × biloks O) = 0  

(2 × (+1)) + biloks Cr + (7 × (-2)) = 0  

(+2) + biloks Cr + (-14) = 0  

biloks Cr + (-12) = 0  

biloks Cr = +12

4. Biloks O pada Na₂O₂

biloks Na = +1 (merupakan unsur golongan IA)  

aturan biloks :  

Bilangan oksidasi unsur golongan utama (IA,IIA,IIIA,IVA,VA,VIA) sesuai dengan golongannya. Contoh : IA = +1, IIA = +2, IIIA = +3, IVA = -4, VA = -3 , VIA = -2 , VIIA = -1 , VIIIA = 0.  

biloks Na₂O₂ = 0  

(2 × biloks Na) + (2 × biloks O) = 0  

(2 × (+1)) + (2 × biloks O) = 0  

(+2) + (2 × biloks O) = 0  

(2 × biloks O) = -2

biloks O = [tex]\frac{-2}{2}[/tex]

biloks O = -1

Sebelum kita mencari bilangan oksidasi dari atom unsur tertentu pada molekul senyawa atau ion di atas, perlu untuk mengingat kembali beberapa aturan bilangan oksidasi atom unsur atau ion, sebagai berikut.

[1]. Bilangan oksidasi unsur golongan IA (logam alkali) adalah +1.

[2]. Bilangan oksidasi unsur O (oksigen) adalah -2.

[3]. Bilangan oksidasi unsur H (hidrogen) adalah +1, kecuali atom hidrogen yang berikatan langsung dengan logam (disebut logam hidrida) memiliki bilangan oksidasi -1.

[4]. Bilangan oksidasi unsur golongan VIIA (halogen, non logam) yang tidak terikat dengan O (oksigen) adalah -1.

[5]. Senyawa ion memiliki bilangan oksidasi total sesuai muatannya,

[6]. Total bilangan oksidasi senyawa netral adalah nol.

Pengerjaan

[a]. HCl

Sesuai dengan aturan nomor 4, 5, dan 7 di atas, dapat disimpulkan:

\boxed{bilangan~oksidasi~H = +1}

bilangan oksidasi H=+1

\boxed{bilangan~oksidasi~Cl = -1}

bilangan oksidasi Cl=−1

Total bilangan oksidasi senyawa HCl sebagai netral adalah nol.

[b]. N₂O₅

Sesuai dengan aturan nomor 2 di atas, bilangan oksidasi O adalah -2, maka kita cari bilangan oksidasi N. Ingat, total bilangan oksidasi senyawa netral adalah nol.

2[biloks N] + 5[biloks O] = 0

2(biloks N) + 5(-2) = 0

2(biloks N) = 10 ⇒ \boxed{Bilangan~oksidasi~N = +5}

Bilangan oksidasi N=+5

[c]. K₂Cr₂O₇

Sesuai dengan aturan nomor 1 dan 2 di atas, karena K anggota golongan IA maka bilangan oksidasi K adalah +1, dan bilangan oksidasi O tetap -2, jadi kita cari bilangan oksidasi Cr. Ingat, total bilangan oksidasi senyawa netral adalah nol.

2[biloks K] + 2[biloks Cr] + 7[biloks O] = 0

2(+1) + 2(biloks Cr) + 7(-2) = 0

2(biloks Cr) = 14 - 2 ⇒ \boxed{Bilangan~oksidasi~Cr = +6}

Bilangan oksidasi Cr=+6

22. Jika diketahui reaksi redoks berikut Pernyataan yang tepat terkait reaksi redoks 2Al + 6HCl --> 2AlCl3 + 3H2 tersebut adalah… a Al mengalami oksidasi karena biloksnya menurun b HCl mengalami reduksi karena biloksnya meningkat c Al mengalami oksidasi karena biloksnya meningkat d Al mengalami oksidasi karena biloksnya tetap e HCl mengalami oksidasi karena biloksnya menurun


Pernyataan yang tepat terkait reaksi redoks 2Al + 6HCl → 2AlCl₃ + 3H₂ tersebut adalah Al mengalami oksidasi karena biloksnya meningkat. Penentuan reaksi oksidasi atau pun reduksi pada soal ini menggunakan konsep perubahan biloks (bilangan oksidasi). Reaksi oksidasi terjadi peningkatan bilangan oksidasi, sedangkan reaksi reduksi terjadi penurunan bilangan oksidasi.

Pembahasan

Perkembangan teori reaksi oksidasi dan reaksi reduksi adalah:

A.Berdasarkan pelepasan atau pengikatan oksigen

Reaksi redoks diartikan sebagai reaksi pelepasan dan pengikatan elektron, yaitu:

Reaksi oksidasi: reaksi yang melibatkan pengikatan oksigenReaksi reduksi: reaksi yang melibatkan pelepasan oksigen.

B.Berdasarkan pengikatan atau pelepasan elektron

Konsep reaksi redoks semakin berkembang, dan muculah konsep redoks berdasarkan pengingkatan dan pelepasan elektrron.

Reaksi oksidasi: reaksi yang melibatkan pelepasan elektronReaksi reduksi: reaksi yang melibatkan pengikatan elektron

C.Berdasarkan bilangan oksidasi atau biloks

Konsep penentuan bilangan oksidasi (biloks) adalah kesepakatan para ahli. Pada sistem periodik unsur, unsur-unsur logam memiliki biloks positif sedangkan unsur-unsur non logam memiliki biloks negatif.

Reaksi oksidasi: reaksi yang terjadi karena kenaikan biloksReaksi reduksi: reaksi yang terjadi karena penurunan biloks

Adapun penentuan biloks dari tiap-tiap unsur mengikuti aturan sebagai berikut.

1.Biloks= 0, dimiliki oleh:

unsur-unsur bebas (contoh: Na, Mg, dan Ag),molekul dwi atom (H₂, Cl₂, dan N₂),molekul poli atom (S₈ dan P₄),Molekul netral (KMnO₄, H₂S, dan CuO)

2.Biloks= +1, dimiliki oleh golongan IA (logam alkali)

3.Biloks= +2, dimiliki oleh golongan IIA (alkali tanah)

4.Biloks ion sesuai dengan muatannya.

Na⁺ = +1Mg²⁺= +2SO₄²⁻= -2PO₄³⁻= -3

5.Biloks H= +1, kecuali pada:

senyawa hidrida (NaH dan CaH₂) biloks H adalah -1

6. Biloks O= -2, kecuali pada:

Senyawa peroksida (H₂O₂), biloks= -1Senyawa superoksida (KO₂), biloks= -½Senyawa OF₂, biloks= +2

Nah, untuk kasus soal di atas diselesaikan dengan konsep reaksi redoks berdasarkan perubahan bilangan oksidasi atau biloks. Persamaan reaksi yang diketahui adalah:

2Al + 6HCl → 2AlCl₃ + 3H₂

Untuk mengetahui pernyataan mana yang benar, caranya adalah:

1.Menentukan bilangan oksidasi masing-masing unsur

Al adalah unsur bebas, maka biloks Al=0

HCl adalah senyawa netral, maka biloks HCl= 0

H golongan IA, maka biloks H= +1Biloks Cl= -1

AlCl₃ adalah senyawa netral, maka biloks AlCl₃= 0

Al golongan IIIA, maka biloks = +3Biloks Cl= -1H₂ adalah molekul diatomik, maka biloks H₂= 0

2.Menentukan reaksi oksidasi dan reaksi reduksi

Berdasarkan uraian bilangan oksidasi masing-masing unsur, maka yang mengalami reaksi oksidasi, yaitu kenaikan bilangan oksidasi adalah Al - AlCl₃, yaitu dari biloks 0 menjadi +3. Untuk unsur yang mengalami reaksi reduksi, yaitu penurunan bilangan oksidasi adalah HCl - H₂, yaitu perubahan biloks +1 menjadi 0.

Berdasarkan penjelasan di atas, maka dapat disimpulkan pernyataan yang benar adalah:

Al mengalami oksidasi karena biloksnya meningkat.HCl mengalami reduksi karena biloksnya menurun.

27. K2CrO7, berapa bilangan biloks Cr? CaSO4, berapa biloks S? NaNO3 , berapa biloks N ? NH4+ Berapa biloks N ? SO2, berapa biloks S?


1)  Bilangan biloks Cr dalam K₂CrO₇ adalah +12.

2) Bilangan biloks S dalam CaSO₄ adalah +6.

3) Bilangan biloks N dalam NaNO₃ adalah +5.

4) Bilangan biloks N dalam NH₄⁺ adalah -3.

5) Bilangan biloks S dalam SO₂ adalah +4.

Pembahasan : 1)  Bilangan biloks Cr dalam K₂CrO₇

biloks K = +1 (Golongan IA)

aturan biloks :  

Bilangan oksidasi unsur golongan utama (IA,IIA,IIIA,IVA,VA,VIA ) sesuai dengan golongannya. Contoh : IA = +1, IIA = +2, IIIA = +3, IVA = -4, VA = -3 , VIA = -2 , VIIA = -1 , VIIIA = 0.

biloks O = -2  

aturan biloks :  

Bilangan oksidasi dalam senyawa O adalah -2 kecuali dalam peroksida (-1) dan dalam senyawa biner dengan fluor (+2). Contoh : Bilangan oksidasi O dalam H₂O = -2 , kecuali bilangan oksidasi O dalam H₂O₂ dan BaO₂ = -1 , dan bilangan oksidasi O dalam OF = +2.

biloks K₂CrO₇ = 0  

(2 × biloks K) + biloks Cr + (7 × biloks O) = 0

(2 × (+1)) + biloks Cr + (7 × (-2)) = 0

(+2) + biloks Cr + (-14) = 0

biloks Cr + (-12) = 0

biloks Cr = +12

2) Bilangan biloks S dalam CaSO₄

biloks Ca = +2 (Golongan IIA)

aturan biloks :  

Bilangan oksidasi unsur golongan utama (IA,IIA,IIIA,IVA,VA,VIA ) sesuai dengan golongannya. Contoh : IA = +1, IIA = +2, IIIA = +3, IVA = -4, VA = -3 , VIA = -2 , VIIA = -1 , VIIIA = 0.

biloks O = -2  

aturan biloks :  

Bilangan oksidasi dalam senyawa O adalah -2 kecuali dalam peroksida (-1) dan dalam senyawa biner dengan fluor (+2). Contoh : Bilangan oksidasi O dalam H₂O = -2 , kecuali bilangan oksidasi O dalam H₂O₂ dan BaO₂ = -1 , dan bilangan oksidasi O dalam OF = +2.

biloks CaSO₄ = 0  

biloks Ca + biloks S + (4 × biloks O) = 0

(+2) + biloks S + (4 × (-2)) = 0

(+2) + biloks S + (-8) = 0

biloks S + (-6) = 0

biloks S = +6

3) Bilangan biloks N dalam NaNO₃

biloks Na = +1 (Golongan IA)

aturan biloks :  

Bilangan oksidasi unsur golongan utama (IA,IIA,IIIA,IVA,VA,VIA ) sesuai dengan golongannya. Contoh : IA = +1, IIA = +2, IIIA = +3, IVA = -4, VA = -3 , VIA = -2 , VIIA = -1 , VIIIA = 0.

biloks O = -2  

aturan biloks :  

Bilangan oksidasi dalam senyawa O adalah -2 kecuali dalam peroksida (-1) dan dalam senyawa biner dengan fluor (+2). Contoh : Bilangan oksidasi O dalam H₂O = -2 , kecuali bilangan oksidasi O dalam H₂O₂ dan BaO₂ = -1 , dan bilangan oksidasi O dalam OF = +2.

biloks NaNO₃ = 0  

biloks Na + biloks N + (3 × biloks O) = 0

(+1) + biloks N + (3 × (-2)) = 0

(+1) + biloks N + (-6) = 0

biloks N + (-5) = 0

biloks N = +5

4) Bilangan biloks N dalam NH₄⁺

biloks H = +1

aturan biloks :  

Bilangan oksidasi hidrogen dalam senyawanya adalah +1 kecuali dalam Hidrida, atom Hidrogen biloksnya adalah -1. Contoh : Bilangan oksidasi H dalam H₂O, NH₃, dan HCl = +1, kecuali Bilangan oksidasi H dalam NaH dan CaH₂ = -1.

biloks NH₄⁺ = +1

biloks N + (4 × biloks H)= +1

biloks N + (4 × (+1)) = +1

biloks N + (+4) = +1

biloks N + (+4) = +1

biloks N = (+1) + (-4)

biloks N = -3

5) Bilangan biloks S dalam SO₂

Biloks O = -2  

aturan biloks :  

Bilangan oksidasi dalam senyawa O adalah -2 kecuali dalam peroksida (-1) dan dalam senyawa biner dengan fluor (+2). Contoh : Bilangan oksidasi O dalam H₂O = -2 , kecuali bilangan oksidasi O dalam H₂O₂ dan BaO₂ = -1 , dan bilangan oksidasi O dalam OF = +2.

biloks SO₂ = 0  

biloks S + (2 × biloks O) = 0

biloks S + (2 × (-2)) = 0

biloks S + (-4) = 0

biloks S = +4
