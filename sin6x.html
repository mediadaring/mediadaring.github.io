11. nilai lim x mendekati 0 6x-sin3x / sin6x + sin4x adalahsekarang please ​


Jawaban:

Untuk mencari nilai dari lim x mendekati 0 dari ekspresi 6x-sin3x / sin6x + sin4x, kita dapat menggunakan aturan dalam kalkulus yang disebut aturan L'Hôpital. Aturan ini memungkinkan kita untuk menghitung batas dari sebuah fungsi ketika nilai x mendekati suatu titik tertentu.

Pertama, kita akan menghitung turunan dari fungsi yang ada di pembilang dan penyebut pecahan. Menggunakan aturan rantai, kita dapat menghitung turunan dari fungsi-fungsi tersebut:

Turunan pembilang: (6 - 3cos3x)

Turunan penyebut: (6cos6x + 4cos4x)

Selanjutnya, kita akan mencari nilai dari batas ketika x mendekati 0 dengan menggunakan turunan yang telah dihitung:

lim x mendekati 0 (6 - 3cos3x) / (6cos6x + 4cos4x)

Menggantikan x dengan 0 dalam turunan, kita akan mendapatkan:

lim x mendekati 0 (6 - 3cos(3*0)) / (6cos(6*0) + 4cos(4*0))

= lim x mendekati 0 (6 - 3cos0) / (6cos0 + 4cos0)

= lim x mendekati 0 (6 - 3) / (6 + 4)

= 3 / 10

Jadi, nilai dari lim x mendekati 0 dari ekspresi 6x-sin3x / sin6x + sin4x adalah 3/10.

36. sederhanakan lah bentuk sin6x+sin4x------------------ !cos6x+cos4x​


[tex]Bentuk~sederhana~dari~\frac{sin6x+sin4x}{cos6x+cos4x}~adalah~tan5x[/tex]

PEMBAHASAN

Trigonometri merupakan ilmu matematika yang mempelajari hubungan antara panjang dan sudut pada segitiga.

Dalam trigonometri terdapat banyak rumus identitas, antara lain untuk rumus pejumlahan dan pengurangan sin dan cos :

[tex]sinx+siny=2sin(\frac{x+y}{2})cos(\frac{x-y}{2})\\\\sinx-siny=2cos(\frac{x+y}{2})sin(\frac{x-y}{2})\\\\cosx+cosy=2cos(\frac{x+y}{2})cos(\frac{x-y}{2})\\\\cosx-cosy=-2sin(\frac{x+y}{2})sin(\frac{x-y}{2})\\[/tex]

.

DIKETAHUI

[tex]\frac{sin6x+sin4x}{cos6x+cos4x}[/tex]

.

DITANYA

Sederhanakan bentuk fungsi tersebut.

.

PENYELESAIAN

[tex]\frac{sin6x+sin4x}{cos6x+cos4x} \\\\=\frac{2sin(\frac{6x+4x}{2})cos(\frac{6x-4x}{2})}{2cos(\frac{6x+4x}{2})cos(\frac{6x-4x}{2})}\\\\=\frac{sin5x.cosx}{cos5x.cosx}\\\\=\frac{sin5x}{cos5x}\\\\=tan5x[/tex]

.

KESIMPULAN

[tex]Bentuk~sederhana~dari~\frac{sin6x+sin4x}{cos6x+cos4x}~adalah~tan5x[/tex]

.
