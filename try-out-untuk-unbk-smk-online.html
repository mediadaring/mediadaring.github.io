16. Tolong Kakak ini untuk bahan try out


1. Bangun : Persegi Panjang
 Luas = panjang x lebar
= 25 x 12
= 300cm²
Keliling = 2 x ( panjang + lebar )
= 2 x (25 + 12)
= 2 x 37
= 74cm

2. Bangun: Segitiga
Luas = 1/2 x alas x tinggi
= 1/2 x 35 x 30
= 1/2 x 1.050
= 525cm²
Keliling = a + b + c
= 30 + 35 + 40
= 105cm

3. Bangun: Trapesium
Luas = 1/2 x (a+b) x t
= 1/2 x (25+28) x 20
= 1/2 x 53 x 20
= 1/2 x 1.060
= 530cm²
Keliling = a+b+c+d
= 25 + 28 + 20 + 30
= 103cm

4. Jajargenjang
Luas = alas x tinggi
= 28 x 30
= 840cm²
Keliling = 2 x (a+t)
= 2 x (28 + 30)
= 2 x 58
= 116cm

5. Belah Ketupat
Luas = 1/2 x d1 x d2
= 1/2 x 8 x 6
= 1/2 x 48
= 24cm
Keliling = s+s+s+s
= 8+8+6+6
= 28cm

23. Mosi unbk harus dilaksanakan di seluruh sma/smk


Mosi: UNBK (Ujian Nasional Berbasis Komputer) harus dilaksanakan di seluruh SMA/ SMK

Pendapat Pro:

Setuju, sebab bila UNBK dilaksanakan maka seluruh SMA/ SMK dapat menghemat kertas. Hal ini dapat pula memangkas biaya kegiatan Ujian Nasional (dapat meringankan beban orang tua murid).

Pendapat Kontra:

Tidak setuju, sebab tidak semua SMA/ SMK memiliki fasilitas komputer yang layak. Untuk menunjang kelancaran kegiatan UNBK sudah seharusnya memakai komputer dengan spesifikasi yang memadai. Sedangkan seluruh SMA/ SMK belum tentu memilikinya (dapat membebani pihak sekolah, bila UNBK dilaksanakan).
