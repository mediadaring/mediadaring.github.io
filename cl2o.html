24. bilangan oksidasi Cl dalam senyawa NaCl, Cl2O, HClO3 ??


Kelas           : X
Mapel          : Kimia
Kategori      : Redoks
Kata Kunci  : bilangan, oksidasi, biloks, Cl, klor, senyawa, natrium, oksida, asam, klorat

Kode : 10.7.9 [Kelas 10 Kimia Bab 9 Redoks]

Teks soal
Berapakah bilangan oksidasi (biloks) Cl dalam senyawa NaCl, Cl₂O, dan HClO₃ ?

Penyelesaian

Silakan pelajari kembali aturan mengenai bilangan oksidasi pada link ini https://brainly.co.id/tugas/14185926

Berikut bilangan-bilangan oksidasi Cl, yakni: -1, +1, +3, +5, dan +7.

(a). NaCl (natrium klorida)

⇒ Bilangan oksidasi Na = +1, sesuai anggota golongan IA
⇒ Total bilangan oksidasi senyawa netral adalah nol
⇔ Biloks Na + biloks Cl = 0
⇔ (+1) + biloks Cl = 0
Diperoleh bilangan oksidasi Cl = -1

(b). Cl₂O (dikloro monoksida)

⇒ Bilangan oksidasi O = -2
⇒ Total bilangan oksidasi senyawa netral adalah nol
⇔ 2(biloks Cl) + biloks O = 0
⇔ 2(biloks Cl) + (-2) = 0
⇔ 2(biloks Cl) = 2
Diperoleh bilangan oksidasi Cl = +1

(c). HClO₃ (asam klorit)

⇒ Bilangan oksidasi H = +1
⇒ Bilangan oksidasi O = -2
⇒ Total bilangan oksidasi senyawa netral adalah nol
⇔ Biloks H + biloks Cl + 3(biloks O) = 0
⇔ (+1) + biloks Cl + 3(-2) = 0
⇔ Biloks Cl = 6 - 1
Diperoleh bilangan oksidasi Cl = +5
