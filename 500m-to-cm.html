15. skala pada denah lapangan sekolah adalah 1 : 1.000. lebar yang sebenarnya 500m. lebar pada denah adalah ..... cm.


Mapel : Matematika
Bab : Skala

Diketahui :
- Skala (S) = 1 : 1.000 = 1/1.000
- Lebar Sebenarnya (LS) = 500 m

Ditanya :
- Lebar Pada Denah (LPD) = ......?

Penyelesaian :
- Lebar Pada Denah (LPD) :
LPD = LS × S
LPD = 500 m × 1/1.000
LPD = 50.000 cm × 1/1.000
LPD = 50.000/1.000 cm
LPD = 50 cm

24. 500m²=........cm²bantu jawab ka ​

Penjelasan dengan langkah-langkah:

500m²=........cm²

Maka*

= 5.000.000 cm²

~Konversi Satuan

_________________

Soal

Konversi dari 500 m² ke cm² adalah ...

» Penyelesaian Soal

Untuk mempermudah mengkonversikan satuan ke satuan lainnya dapat menggunakan Tangga satuan panjang sebagai berikut.

km² ( kilometer persegi )

    hm² ( hektometer persegi )

        dam² ( dekameter persegi )

               m² ( meter persegi )

                  dm² ( desimeter persegi )

                       cm² ( centimeter persegi )

                           mm² ( milimeter persegi )

Keterangan

Naik 1 Tangga ➞ Dibagi 100 ( ÷ 100 )Turun 1 Tangga ➞ Dikali 100 ( × 100 )

———

✧ Penyelesaian

500 m² → cm²

→ 500 × 10.000

→ 5.000.000 cm²

• Kesimpulan

Jadi, Konversi dari 500 m² ke cm² adalah 5.000.000 cm²
