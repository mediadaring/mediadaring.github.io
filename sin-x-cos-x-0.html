17. jika sin x cos y - 7 cos x sin y = 0maka nilai sin(x + y)/sin (x - y) =?​


Jika sin x cos y – 7 cos x sin y = 0, maka:
sin(x + y) / sin(x – y) = 4/3.
 

Penjelasan dengan langkah-langkah:

CARA 1

Pertama-tama, dapat kita perhatikan bahwa

[tex]\begin{aligned}&\sin x \cos y - 7 \cos x \sin y = 0\\&\Rightarrow \sin x\cos y-\cos x\sin y-6\cos x\sin y=0\\&\Rightarrow \sin x\cos y-\cos x\sin y=6\cos x\sin y\\&\Rightarrow \sin(x-y)=6\cos x\sin y\quad...(1)\end{aligned}[/tex]

Kemudian,

[tex]\begin{aligned}&\sin x \cos y - 7 \cos x \sin y = 0\\&\Rightarrow \sin x\cos y+\cos x\sin y-8\cos x\sin y=0\\&\Rightarrow \sin x\cos y+\cos x\sin y=8\cos x\sin y\\&\Rightarrow \sin(x+y)=8\cos x\sin y\quad...(2)\end{aligned}[/tex]

Maka, berdasarkan persamaan (1) dan (2), dapat kita peroleh:

[tex]\begin{aligned}\frac{\sin(x+y)}{\sin(x-y)}&=\frac{8\cancel{\cos x\sin y}}{6\cancel{\cos x\sin y}}\\&=\frac{8}{6}=\frac{\cancel{2}\cdot4}{\cancel{2}\cdot3}\\\therefore\ \frac{\sin(x+y)}{\sin(x-y)}&=\boxed{\,\bf\frac{4}{3}\,}\end{aligned}[/tex]
[tex]\blacksquare[/tex]

CARA 2

Perhatikan bahwa

[tex]\begin{aligned}&\sin x \cos y - 7 \cos x \sin y = 0\\&\Rightarrow \sin x \cos y=7 \cos x \sin y\end{aligned}[/tex]

Maka:

[tex]\begin{aligned}\frac{\sin(x+y)}{\sin(x-y)}&=\frac{\sin x\cos y+\cos x\sin y}{\sin x\cos y-\cos x\sin y}\\&=\frac{7\cos x\sin y+\cos x\sin y}{7\cos x\sin y-\cos x\sin y}\\&=\frac{8\cancel{\cos x\sin y}}{6\cancel{\cos x\sin y}}\\&=\frac{8}{6}=\frac{\cancel{2}\cdot4}{\cancel{2}\cdot3}\\\therefore\ \frac{\sin(x+y)}{\sin(x-y)}&=\boxed{\,\bf\frac{4}{3}\,}\end{aligned}[/tex]
[tex]\blacksquare[/tex]

31. Buktikan bahwa sin x - sin y/ cos x + cos y + cos x - cos y/sin x + sin y = 0


pakai tanda kurungnya, biar lebih jelas.

(sinx - siny) / (cosx + cosy) + (cosx - cosy) / (sinx + siny)

= ((sinx - siny)(sinx + siny) + (cosx - cosy)(cosx + cosy)) / (cosx + cosy)(sinx + siny)

= ((sin²x - sin²y) + (cos²x - cos²y)) / (cosx + cosy)(sinx + siny)
= (sin²x + cos²x - sin²y - cos²y) / (cosx + cosy)(sinx + siny)
= ((sin²x + cos²x) - (sin²y + cos²y)) / (cosx + cosy)(sinx + siny)
= (1 - 1) / (cosx + cosy)(sinx + siny)
= 0 / (cosx + cosy)(sinx + siny)
