21. Apa Keuntungan Para Pembuat Aplikasi Dalam Membuat Aplikasi?​


Keuntungan para pembuat aplikasi dalam membuat aplikasi, adalah...

Pembahasan :

Dalam membuat suatu aplikasi, menjadi salah satu fokus utama sebuah perusahaan atau pembisnis untuk melebarkan sayap bisnisnya.

Dengan adanya Application Store, ini dapat membuat nisia aplikasi dapat semakin penting dalam ekosistem bisnis karena dapat menopang bisnis melalui aplikasi yang dikembangkan pembisnis atau sebuah perusahaan. Berikut adalah cara-cara yang umum para pembuat aplikasi dalam membuat aplikasi yaitu :

1. Menjual Aplikasinya terhadap klien

2. Menjual Aplikasinya terhadap pengguna application store

3. Menerapkan harga atau biaya yang memungkinkan penggunanya berlangganan pada aplikasi tersebut

4. Menerapkan pembayaran/membayar in-app atau In-app Purchase

5. Menerapkan Mesin iklan, karena melalui mesin iklan memungkinkan memiliki nilai yang fantastis

6. Sponsor dari perusahaan atau badan marketing membantu aplikasi kamu jadi populer

Keuntungannya adalah :

Mendapat penghasilan yang berasal dari App Store / Application Store tempat aplikasi dipasarkan.

40. sebutkan 3 aplikasi pengolah kata pada komputer jelaskan pula kelebihan tiap apliaksi dibandingkan dengan aplikasi lainnya ? ​


Jawaban:

Misalkan pada komputer dg sistem operasi Windows:

1. Notepad

2. Wordpad

3. Ms. Word

Penjelasan:

1. Kelebihan Notepad ialah:

- aplikasi pengolah kata yg sangat ringan

- tidak membutuhkan kapasitas harddisk yg besar

- tidak perlu memory komputer yg tinggi.

2. Kelebihan Wordpad ialah:

- aplikasi pengolah kata yg cukup sederhana karena ada beberapa fitur seperti Ms. Word

- mampu membuka dan menyimpan data dg format .doc

- pemakaiannya mudah karena memiliki interface (antar muka) mirip Ms. Word.

3. Kelebihan Ms. Word ialah:

- aplikasi pengolah kata yg sangat lengkap fiturnya.

- Pada versi terbaru, mendukung penyimpanan data dg format PDF

- dokumen yg disimpan, dapat diamankan memakai password

Amatilah lampiran:

1. Notepad ditunjukkan tanda panah merah

2. Wordpad ditunjukkan tanda panah hijau

3. Ms. Word ditunjukkan tanda panah kuning
