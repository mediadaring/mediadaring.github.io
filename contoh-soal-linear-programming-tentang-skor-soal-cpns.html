4. dalam suatu tes ditentukan skor penilaian. jika setiap soal dijawab benar skor 4, skor -2 jawaban salah, skor -1 untuk soal yg tidak di jawab. dari 50 soal, rara dapat menjawab sebanyak 42 soal, skor yang diperoleh rara 82 .banyak soal yang dijawab benar oleh rara adalah. . . buah soal​


Jawab:

Soal yang dijawab benar = 29 soal

Penjelasan dengan langkah-langkah:

Jumlah soal = 50

Aturan :

Dijawab benar = 4

Dijawab salah = -2

Tidak dijawab = -1

Soal Dijawab = 42 soal

Tidak dijawab = 50 - 42 = 8 soal

Skor = 82

Ditanya :

Berapa soal dijawab benar?

Pembahasan :

Misalkan soal dijawab benar = x

82 = 4.x + (-2).(42 - x) + (-1).8

82 = 4x - 84 + 2x - 8

82 = 6x - 92

82 + 92 = 6x

6x = 174

x = 174 : 6 = 29

Jadi, Banyak soal yang dijawab benar adalah 29 soal

Jawaban:

D. 80

Penjelasan dengan langkah-langkah:

jarak = 500 km

kecepatan = x km/jam

waktu = w + 3/4

jarak = kecepatan x waktu

500 = x (w + 3/4)

500 = xw + 3/4x ........ persamaan I

jarak = 400

kecepatan = x km/jam

w = w - 1/2

400 = x (w-1/2)

400 = xw -1/2x .............. persamaan II

persamaan I - Persamaan II (metode eliminasi xw)

100 = 3/4x - (-1/2x)

100 = 3/4x + 1/2x

100 = (3+2) /4 x

100 = 5/4 x

x = 100 x 4/5

x = 80 km/jam (D)

s1 = 500 km

w1 = t + 3/4 jam

s2 = 400 km

w2 = t - 1/2 jam

s1/w1 = s2/w2

500/(t + 3/4) = 400/(t - 1/2)

500(t - 1/2) = 400(t + 3/4)

500t - 250 = 400t + 300

500t - 400t = 300 + 250

100t = 550

t = 550/100

t = 5,5 jam

v1 = s1/w1

v1 = s1/(t + 3/4)

v1 = 500/(5,5 + 3/4)

v1 = 500/(5,5 + 0,75)

v1 = 500/6,25

v1 = 80

jadi x adalah 80 km/jam (D)
