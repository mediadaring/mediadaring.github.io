22. jika y= sec x buktikan bahwa y d²y/dx²=(dy/dx)²+y^4


Materi Persamaan Diferensial

y = sec x

Untuk membuktikan persamaan diferensial tersebut, kita tentukan dahulu turunannya.
dy / dx = sec x tan x
d²y / dx² = sec x tan x tan x + sec x sec²x = sec x tan²x + sec³x

y d²y / dx² = sec x(sec x tan²x + sec³x) = sec²x tan²x + sec⁴x...(1)

(dy / dx)²+y⁴ = sec²x tan²x + sec⁴x...(2)

Karena persamaan (1) dan (2) mempunyai nilai yang sama, maka :
Terbukti bahwa untuk y = sec x memenuhi persamaan y d²y / dx² = (dy / dx)²+y⁴.
