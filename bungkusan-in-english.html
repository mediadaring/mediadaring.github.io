Harga untuk sebuah buku tulis adalah x rupiah,

Harga untuk sebuah pensil adalah y rupiah dan

Harga untuk sebuah penghapus adalah z rupiah.

2x + y + 3z = 21.100.......(1)

2x + y + 2z = 18.100.......(2)

x + 2y + z = 12.800........(3)

yaitu merupakan SPLTV dnegan variabel x, y, dan z.

Penyelesaian SPLTV itu dapat ditentukan dengan metode subtitusi, metode eliminasi atau gabungan keduanya.

- eliminasi variabel y (karena bentuknya paling sederhana)

persamaan (1) dan (2)

2x + y + 3z = 21.100

2x + y + 2z = 18.100 ( - )

z = 3.000

persamaan (2) dan (3), samakan terlebih dahulu koefisien y nya

2x + y + 2z = 18.100 | x2 |

x + 2y + z = 12.800 | x1 |

4x + 2y + 4z = 36.200

x + 2y + z = 12.800 ( - )

3x + 3z = 23.400.....(4)

- substitusi kan nilai z = 3.000 ke persamaan (4)

3x + 3z = 23.400

3x + 3(3.000) = 23.400

3x + 9.000 = 23.400

3x = 23.400 - 9.000

3x = 14.400

x = 4.800

- substitusi kan nilai x = 4.800 dan z = 3.000 ke persamaan (1)

2x + y + 3z = 21.100

2(4.800) + y + 3(3.000) = 21.100

9.600 + y + 9.000 = 21.100

y + 18.600 = 21.100

y = 21.100 - 18.600

y = 2.500

dengan demikian diperoleh

x = 4.800

y = 2.500

z = 3.000

persamaan Sisil(S)

x + y + z = S

substitusi kan nilai x,y, dan z

x + y + z = S

4.800 + 2.500 + 3.000 = S

S = 10.300

jadi uang yang harus dikeluarkan Sisil adalah 10.300
