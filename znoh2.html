6. langkah -langkah dalam menyetarakan a. zn +no3^- -> zno2^2- + nh3 suasana basab. cl2 +io- -> cl- +io4^- suasana basa


Kelas         : XII
Pelajaran   : Redoks & Elektrokimia
Kategori     : Penyetaraan Reaksi
Kata Kunci : metode setengah reaksi, suasana basa, suasana asam, perubahan bilangan oksidasi, seimbang

Pembahasan:

Kali ini kita hanya akan membahas penyetaraan reaksi dengan menggunakan metode setengah reaksi.

Langkah-langkah penyetaraan reaksi metode setengah reaksi
[1]. Selidiki unsur yang mengalami perubahan bilangan oksidasi
[2]. Posisikan unsur-unsur sesuai reaksinya, apabila terjadi reduksi harus ditempatkan di katoda dan kalau terjadi oksidasi tentu harus berlangsung di anoda 
[3]. Samakan koefisien unsur yang mengalami perubahan bilangan oksidasi
[4.a] Suasana ASAM
⇒ Tambahkan H₂O di bagian ruas yang kekurangan atom oksigen, jumlah H₂O sesuai jumlah kekurangan oksigen
⇒ Tambahkan ion H⁺ di bagian ruas yang berlawanan agar jumlah H menjadi seimbang
⇒ Cek muatan di kedua ruas, muatan yang lebih positif ditambahkan sejumlah elektron (sebagai pengurang muatan) sehingga tercapai keseimbangan muatan di kedua ruas
[4.b] Suasana BASA
⇒ Tambahkan H₂O di bagian ruas yang kelebihan atom oksigen
⇒ Tambahkan ion OH⁻ di bagian ruas yang berlawanan; sesuaikan jumlah OH⁻ dan H₂O agar jumlah O dan H di kedua ruas menjadi seimbang
⇒ Cek muatan di kedua ruas, muatan yang lebih positif ditambahkan sejumlah elektron (sebagai pengurang muatan) sehingga tercapai keseimbangan muatan di kedua ruas
[5]. Samakan jumlah elektron dari kedua setengah reaksi tersebut, agar jumlah elektron yang dilepaskan pada setengah reaksi oksidasi tepat sama dengan jumlah elektron yang diikat pada setengah reaksi reduksi
[6]. Susun kembali kedua setengah reaksi menjadi persamaan reaksi yang utuh seperti semula

[Soal-a]
Zn + NO₃⁻ → ZnO₂²⁻ + NH₃     (dalam suasana basa)
Periksa bilangan oksidasi:

Bilangan oksidasi Zn di ruas kiri = 0
Bilangan oksidasi Zn di ruas kanan = +2, karena (+2) + 2(-2) = -2
Terjadi peristiwa reduksi karena terdapat penurunan bilangan oksidasi

Bilangan oksidasi N di ruas kiri = +5, karena (+5) + 3(-2) = -1
Bilangan oksidasi N di ruas kanan = -3, karena (-3) + 3(+1) = 0
Terjadi peristiwa oksidasi karena terdapat kenaikan bilangan oksidasi

Katoda: Zn + 4OH⁻ → ZnO₂²⁻ +  2H₂O + 2e   (kalikan empat)
Anoda: NO₃⁻ + 6H₂O + 8e → NH₃ + 9OH⁻
---------------------------------------------------------(+)
Redoks: 4Zn + NO₃⁻ + 16OH⁻ + 6H₂O → 4ZnO₂²⁻ + NH₃ + 8H₂O + 9OH⁻

Tahap terakhir, sesuaikan jumlah OH⁻ dan H₂O

Setara
4Zn + NO₃⁻ + 7OH⁻ + → 4ZnO₂²⁻ + NH₃ + 2H₂O
Selesai

[Soal-b]
Cl₂ + IO⁻ → Cl⁻ + IO₄⁻    (dalam suasana basa)
Periksa bilangan oksidasi:

Bilangan oksidasi Cl di ruas kiri = 0
Bilangan oksidasi Cl di ruas kanan = -1
Terjadi peristiwa reduksi karena terdapat penurunan bilangan oksidasi

Bilangan oksidasi I di ruas kiri = +1, karena (+1) + (-2) = -1
Bilangan oksidasi I di ruas kanan = +7, karena (+7) + 4(-2) = -1
Terjadi peristiwa oksidasi karena terdapat kenaikan bilangan oksidasi

Katoda: Cl₂ + 2e → 2Cl⁻     
Anoda: IO⁻ + 6OH⁻ → IO₄⁻ + 3H₂O + 6e  
(dengan munculnya 3H₂O dan 6OH⁻ maka jumlah H dan O di kedua ruas menjadi seimbang)

Katoda: Cl₂ + 2e → 2Cl⁻        (kalikan tiga)
Anoda: IO⁻ + 6OH⁻ → IO₄⁻ + 3H₂O + 6e 
----------------------------------------------------- (+)

Setara
Redoks: 3Cl₂ + IO⁻ + 6OH⁻→ 6Cl⁻ + IO₄⁻ + 3H₂O 
Selesai

9. Setarakan reaksi redoks dengan metode bilangan oksidasi NH3+ZNO2^2 menghasilkan N2O3+ZN


Setarakan reaksi redoks dengan metode bilangan oksidasi

NH₃ + ZnO₂⁻² ==> N₂O₃ + Zn

Jawab:

Sebelum kita mulai melakukan penyetaraan reaksi, kita hitung biloks dari masing-masing unsur terlebih dahulu, yaitu:

Biloks NH₃ = 0

Biloks N + 3 x Biloks H = 0

Biloks N + 3 x (+1) = 0

Biloks N = -3

Biloks ZnO₂⁻² = -2

Biloks Zn + 2 x Biloks O = -2

Biloks Zn + 2 x (-2) = -2

Biloks Zn + (-4) = -2

Biloks Zn = +2

Biloks N₂O₃ = 0

2 x Biloks N + 3 x Biloks O = 0

2 x Biloks N + 3 x (-2) = 0

2 x Biloks N + (-6) = 0

2 x Biloks N = + 6

Biloks N = +3

Biloks Zn = 0

Setelah semua bilangan oksidasi setiap unsur diketahui, maka kita bisa mulai melakukan penyetaraan reaksi dengan cara Bilangan Oksidasi dengan langkah-langkah berikut:

Langkah 1 Menuliskan unsur yang mengalami perubahan bilangan Oksidasi

Reaksi:

NH₃ + ZnO₂⁻² ==> N₂O₃ + Zn

Berdasarkan reaksi di atas diketahui:

Reaksi Reduksi :

ZnO₂⁻² ==> Zn

Biloks Zn dari +2 menjadi 0

Reaksi Oksidasi :

NH₃  ==> N₂O₃

Biloks N dari -3 menjadi +3

Langkah 2 Menyetarakan unsur yang mengalami perubahan Bilangan Oksidasi

Reaksi Reduksi :

ZnO₂⁻² ==> Zn

Atom Zn sudah setara

Reaksi Oksidasi :

NH₃  ==> N₂O₃  menjadi  2 NH₃  ==> N₂O₃

Atom N sudah setara

Langkah 3 Menentukan Jumlah Kenaikan dan Penurunan Bilangan Oksidasi dari unsur yang mengalami Oksidasi dan Reduksi

Reaksi Reduksi :

ZnO₂⁻² ==> Zn

Perubahan biloks Zn dari + 2 menjadi 0

Terjadi penurunan biloks sebesar 2 angka

Reaksi Oksidasi :

2 NH₃  ==> N₂O₃

Perubahan biloks N dari -3 menjadi +3

Terjadi penurunan biloks sebesar 6 angka, tetapi karena atom N yang terlibat sebanyak 2, maka perubahan biloks juga dikalikan 2, sehingga menjadi 12 angka.

Langkah 4 Menyamakan jumlah perubahan Bilangan Oksidasi

Untuk menyamakan jumlah perubahan bilangan oksidasi pada reaksi reduksi dan oksidasi, maka koefisien reaksi pada reaksi reduksi dikalikan dengan nilai kenaikan bilangan oksidasi dari unsur pada reaksi oksidasi.

Sebaliknya koefisien reaksi Oksidasi dikalikan dengan nilai penurunan bilangan oksidasi dari unsur pada reaksi reduksi.

Sehingga koefisien reaksi pada reaksi redoks menjadi:

4 NH₃ + 12 ZnO₂⁻² ==> 2 N₂O₃ + 12 Zn

Langkah 5 Menyamakan Muatan (Karena reaksi berupa Ion)

Jika misalkan reaksi ini berjalan dalam suasan asam, maka penyetaraan dilakukan dengan penambahan ion H⁺, jika suasana basa penyetaraan dengan ion OH⁻.

Jumlah muatan di kiri = 12 x -2 = - 24

Jumlah muatan di kanan = 0

Suasana Asam

Jika reaksi berjalan dalam suasana asam maka kita tambahkan ion H⁺ disisi kiri sebanyak 24 buah, menjadi:

4 NH₃ + 12 ZnO₂⁻² + 24 H⁺ ==> 2 N₂O₃ + 12 Zn

Suasana Basa

Jika reaksi berjalan dalam suasana basa maka kita tambahkan ion OH⁻ disisi kanan sebanyak 24 buah, menjadi:

4 NH₃ + 12 ZnO₂⁻² ==> 2 N₂O₃ + 12 Zn + 24 OH⁻

Langkah 6  Menyetarakan atom H

Untuk menyamakan jumlah atom H maka kita hitung jumlah di sisi kanan dan kiri kemudian disetarakan dengan penambahan molekul H₂O.

Suasana Asam

4 NH₃ + 12 ZnO₂⁻² + 24 H⁺ ==> 2 N₂O₃ + 12 Zn

Atom H di kiri = 12 + 24 = 36

Atom H di kanan = 0

Sehingga di sisi kanan kita tambahkan 18 molekul H₂O, karena dalam 1 molekul H₂O mengandung 2 atom H. Sehingga menjadi:

4 NH₃ + 12 ZnO₂⁻² + 24 H⁺ ==> 2 N₂O₃ + 12 Zn + 18 H₂O (SETARA)

Suasana Basa

4 NH₃ + 12 ZnO₂⁻² ==> 2 N₂O₃ + 12 Zn + 24 OH⁻

Atom H di kiri = 12

Atom H di kanan = 24

Sehingga di sisi kiri kita tambahkan 6 molekul H₂O, karena dalam 1 molekul H₂O mengandung 2 atom H. Sehingga menjadi:

4 NH₃ + 12 ZnO₂⁻² + 6 H₂O ==> 2 N₂O₃ + 12 Zn + 24 OH⁻ (SETARA)

Sisanya kita lakukan pengecekan ulang dengan menghitung ulang jumlah atom-atom di sisi kiri dan kanan.

Jika langkah-langkah kita sudah benar maka jumlah atom Oksigen di kiri dan kanan akan otomatis sama, jika tidak sama maka perlu dicek ulang per langkah kerja.

Penjelasan:

Langkah-langkah penyelesaian dilakukan dengan menggunakan Metode Setengah Reaksi:

Langkah 1 : Menuliskan Reaksi Reduksi dan Oksidasi

Langkah 2 : Menyetarakan unsur yang terlibat dalam reaksi Reduksi dan Oksidasi

Langkah 3 : Menyetarakan Jumlah atom Oksigen dan Hidrogen (Oksigen dahulu)

Langkah 4 : Menyetarakan muatan

Penyetaraan dilakukan dengan penambahan elektron (e⁻)

Langkah 5 : Menyetarakan Elektron

Langkah 6 : Silahkan Cek ulang dan Hitung Ulang jika langkah benar maka sudah SETARA.

Berdasarkan langkah diatas, maka kita bisa coba mengerjakan soal-soal tersebut, yaitu:

As₂O₃ + Zn ==> AsH₃ + ZnO₂

Langkah 1 : Menuliskan Reaksi Reduksi dan Oksidasi

Reduksi : As₂O₃ ==> AsH₃ (Biloks As = +3 ==> -3)

Oksidasi : Zn ==> ZnO₂ (Biloks Zn = 0 ==> +4)

Langkah 2 : Menyetarakan unsur yang terlibat dalam reaksi Reduksi dan Oksidasi

Reduksi : As₂O₃ ==> AsH₃

(As pada AsH₃ kita setarakan dengan menambah koefisien reaksi)

Reduksi : As₂O₃ ==> 2 AsH₃ (Setara)

Oksidasi : Zn ==> ZnO₂ (Zn sudah setara)

Langkah 3 : Menyetarakan Jumlah atom Oksigen dan Hidrogen (Oksigen dahulu)

Penyetaraan atom Oksigen dilakukan dengan penambahan molekul H₂O sesuai dengan kondisi asam, yaitu:

Reduksi : As₂O₃ ==> 2 AsH₃

Karena di sisi kiri terdapat 3 atom Oksigen sedangkan di kanan tidak ada, kita tambahkan 3 molekul H₂O di kanan sehingga jumlah atom O di kiri dan kanan sama, yaitu:

Reduksi : As₂O₃ ==> 2 AsH₃ + 3 H₂O

selanjutnya kita setarakan jumlah atom H dengan menambahkan 12 ion H⁺ disisi kiri sehingga jumlah atom H menjadi sama, yaitu:

Reduksi : As₂O₃ + 12 H⁺ ==> 2 AsH₃ + 3 H₂O  (H & O Setara)

selanjutnya:

Oksidasi : Zn ==> ZnO₂

Karena disisi kanan terdapat 2 atom Oksigen, maka kita tambahkan 2 molekul H₂O di kiri sehingga jumlah atom oksigen menjadi sama, yaitu:

Oksidasi : Zn + 2 H₂O ==> ZnO₂

selanjutnya kita setarakan jumlah atom H dengan menambahkan 4 ion H⁺ disisi kanan sehingga jumlah atom H menjadi sama, yaitu:

Oksidasi : Zn + 2 H₂O ==> ZnO₂ + 4 H⁺ (H & O Setara)

Langkah 4 : Menyetarakan muatan

Penyetaraan dilakukan dengan penambahan elektron (e⁻)

Reduksi : As₂O₃ + 12 H⁺ ==> 2 AsH₃ + 3 H₂O

Muatan di kiri = 0 + (12 x (1)) = + 12

Muatan di kanan = 0

maka kita tambahkan 12 elektron di kiri sehingga jumlah muatan sama yaitu 0,

Reduksi : As₂O₃ + 12 H⁺ + 12 e⁻ ==> 2 AsH₃ + 3 H₂O (Muatan Setara)

Selanjutnya:

Oksidasi : Zn + 2 H₂O ==> ZnO₂ + 4 H⁺

Muatan dikiri = 0

Muatan dikanan = 0 + 4 = +4

maka kita tambahkan 4 elektron di kanan sehingga jumlah muatan sama yaitu 0,

Oksidasi : Zn + 2 H₂O ==> ZnO₂ + 4 H⁺ + 4 e⁻ (Muatan Setara)

Langkah 5 : Menyetarakan Elektron

Menyetarakan elektron dengan melakukan perkalian sehingga jumlah elektron pada reaksi Reduksi dan Oksidasi sama, yaitu:

Reduksi : As₂O₃ + 12 H⁺ + 12 e⁻ ==> 2 AsH₃ + 3 H₂O        |x 1|

Oksidasi : Zn + 2 H₂O ==> ZnO₂ + 4 H⁺ + 4 e⁻                   |x 3|

-------------------------------------------------------------------------------------------------

Reduksi : As₂O₃ + 12 H⁺ + 12 e⁻ ==> 2 AsH₃ + 3 H₂O

Oksidasi : 3 Zn + 6 H₂O ==> 3 ZnO₂ + 12 H⁺ + 12 e⁻

--------------------------------------------------------------------------------------   -

Redoks : As₂O₃ + 3 Zn + 3 H₂O ==> 2 AsH₃ + 3 ZnO₂  (SETARA)

* Karena jumlah elektron adalah sama yaitu 12 elektron, maka kita lakukan pencoretan

** Karena jumlah ion H⁺ adalah sama yaitu 12 ion H⁺, maka kita lakukan pencoretan

*** Karena jumlah molekul H₂O dikiri ada 6 dan di kanan ada 3, maka kita lakukan pencoretan disisi kanan sebanyak 3, sehingga di sisi kiri tersisa 3 molekul H₂O.

Langkah 6 : Silahkan Cek ulang dan Hitung Ulang jika langkah benar maka sudah SETARA.

Pembahasan :

Bilangan Oksidasi

Bilangan oksidasi adalah banyaknya elektron yang dilepas/diterima dalam pembentukan suatu molekul/ion. Nilai biloks dapat bernilai positif/negatif. Bilangan oksidasi  disingkat dengan biloks. Beberapa atom hanya memiliki satu biloks, ada juga yang memiliki lebih dari satu biloks.

Tanda (+) dan (-) pada biloks ditulis sebelum angkanya, misalnya +2, atau +1; sedangkan pada muatan ditulis sesudah angkanya, misalnya 2+ atau 3+. Bilangan oksidasi menunjukkan besarnya muatan yang disumbangkan oleh atom atau unsur tersebut pada molekul atau ion yang dibentuknya.

Aturan Biloks :

Biloks unsur bebas bentuk monoatomik, diatomik, triatomik, tetraatomik, dst, memiliki harga nol. Contoh: Fe, C, H₂, Cl₂, F₂, O₂, P₄, dan S₈. Biloks atom F adalah -1. Atom logam selalu memiliki biloks positif dengan harga sesuai dengan nomor golongannya, kecuali untuk logam transisi yang memiliki biloks lebih dari satu. Biloks atom Li, Na, K, Rb, dan Cs adalah +1. Biloks atom Be, Mg, Ca, Sr, Ba, dan Ra adalah +2. Biloks atom Al adalah +3. Biloks atom H umumnya adalah +1, kecuali jika berikatan dengan unsur logam, seperti Na, biloks menjadi negatif (-1). Senyawa atom H dengan unsur logam disebut senyawa hidrida. Biloks atom O umumnya adalah -2, kecuali jika berikatan dengan atom F, atau dalam senyawaan peroksida dan superoksida. biloks atom O adalah +2 dalam OF₂, dalam senyawa peroksida (misalnya H₂O₂) berharga -1. Adapun dalam senyawa superoksida (seperti KO₂), biloks O bernilai -1/2. Jumlah seluruh biloks atom-atom penyusun satu ion sama dengan muatan ion tersebut. Contoh : S²⁻ = -2;  Fe³⁺ = +3;  MnO₄⁻ = -1;  dan Cr₂O₇²⁻ = -2. Jumlah biloks unsur-unsur pembentuk senyawa netral sama dengan nol.

Reaksi redoks terjadi apabila terjadi serah terima elektron yang disertai dengan perubahan bilangan oksidasi atom-atom yang terlibat reaksi. Reaksi redoks selalu terdiri dari setengah reaksi oksidasi dan setengah reaksi reduksi.

Perbedaan Oksidator dan Reduktor :

Oksidator atau zat pengoksidasi merupakan zat yang mengalami reduksi. Zat ini menyebabkan zat lain mengalami oksidasi. Contoh zat pengoksidasi termasuk halogen, natrium nitrat, dan asam nitrat. Reduktor atau zat pereduksi merupakan zat yang mengalami oksidasi. Zat ini menyebabkan zat lain mengalami reduksi. Contoh-contoh zat pereduksi termasuk alkali  tanah, asam format, dan senyawa sulfit. Autoredoks / disproporsionasi adalah reaksi redoks dimana hanya satu atom yang mengalami oksidasi dan reduksi.
