4. integral 4x^2cos^2x dx =


Int (x^2 cos 2x) dx 

Ingat integral parsial 
Int u dv = uv - int v du 
ambil 
u = x^2 
du = 2x dx 
dv = cos 2x dx 
v = sin 2x 

Int (x^2 cos 2x) dx = x^2 sin 2x - int sin 2x 2x dx 
Int (x^2 cos 2x) dx = x^2 sin 2x - 2 * int x sin 2x dx 

Kita cari bagiannya 
int x sin 2x dx 
Misal u = x, du = dx 
dv = sin 2x, v = - cos 2x 
Jadi: 
int x sin 2x dx = x (-cos 2x) - int -cos 2x dx 
int x sin 2x dx = -x cos 2x + sin 2x + C 

Nah, substitusi dengan persamaan pertama 
Int (x^2 cos 2x) dx = x^2 sin 2x - 2 (-x cos 2x + sin 2x + C)

7. Di ketahui Fx = 2cos pangkat2 2x, turunan dari Fx adalah


gunakan aturan rantai

misal a= (1 - 2x)

da/dx = -2

misal b = cos a

db/da = -sin a

misal c = b³

dc/db = 3b²

misal y = 2c

dy / dc = 2

f'(x) = dy/dc * dc/db * db/da * da/dx

= 2 * 3b² * -sin a * -2

= 12b² sin a

= 12(cos a)² sin a

= 12(cos (1 - 2x))² sin(1 - 2x)

= 12(cos²(1 - 2x))(sin(1 - 2x))

= 12(cos(1-2x).cos(1-2x))(sin(1-2x))

= 6(cos(1-2x).2cos(1-2x))(sin(1-2x))

= 6(cos(1-2x))sin2(1-2x)
