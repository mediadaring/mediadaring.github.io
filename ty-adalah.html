32. maaf saya mau ty lagi​


Diketahui:

(3x-2)/x < x

Ditanyakan:

x

Jawab:

(3x-2)/x < x

((3x-2)/x)-x < 0

((3x-2)/x)-(x²/x) < 0

(-x²+3x-2)/x < 0

(x²-3x+2)/x > 0

Maka:

x²-3x+2 > 0

cari batas

x²-3x+2 = 0

(x-2)(x-1) = 0

x=2 atau x=1

x<1 atau x>2

Agar terdefinisi maka x≠0

Sehingga:

0<x<1 atau x>2
