14. menu pada Delphi 7 sebutkan menu-menu tersebut


Main Menu (Menu Utama)
Menu pada Delphi memiliki kegunaan yang sama seperti menu pada aplikasi windows. Dengan menggunakan fasilitas menu, Anda dapat memanggil atau menyimpan program, sehingga pada dasarnya semua perintah yang dikerjakan dapat ditemukan pada bagian menu utama. Nama-nama menu ditampilkan pada bagian menu bar yang terdapat pada bagian atas layar. Untuk bekerja dengan menggunakan menu, dapat menggunakan fasilitas mouse maupun keyboard.
Toolbar
Delphi memiliki beberapa toolbar yang masing-masing memiliki perbedaan fungsi dan setiap tombol pada bagian toolbar berfungsi untuk menggantikan suatu perintah menu yang sering digunakan. Toolbar terletak pada bagian bawah baris menu. Pada kondisi default, terdapat enam bagian toolbar, antara lain standard, view, debug, desktop, custom dan component palette.
Component palette
Berisi sekumpulan ikon yang melambangkan komponen-komponen yang terdapat pada VCL (Visual Component Library). Pada component palette terdapat beberapa page control seperti standard, additional, Win32, System, Data Access

21. dalam menu bar terdapat menu file dan menu edit apakah fungsi dari ke dua menu


Menu file berfungsi untuk mengolah dokumen secara umum, seperti:

- Membuat dokumen baru (New)

- Membuka dokumen yang sebelumnya telah dibuat (Open)

- Mencetak dokumen (Print)

- Menyimpan dokumen (Save)

- Menutup dokumen (Close)

Menu edit berfungsi untuk mengolah dokumen secara spesifik, seperti:

- Menyalin teks di dalam dokumen (Copy)

- Memotong teks di dalam dokumen (Cut)

- Menempel teks yang sebelumnya telah disalin (Paste)

- Mencari teks tertentu dalam dokumen (Find)

22. menu flyout dalam CorelDRAW merupakan menu-menu dibawah menu


Menu flyout dalam CorelDRAW merupakan menu-menu dibawah menu Toolbar

Untuk membuat Menu Flyout dapat menggunakan langkah-langkah sebagai berikut

- Klik kanan pada Toolbar

- Dari menu konteks yang muncul, klik Customize > [Nama Toolbar] Toolbar > Add New Flyout

- Akan muncul menu Flyout yang masing kosong pada Toolbar

- Tambahkan tool pada menu Flyout dengan cara yang sama seperti menambahkan tool pada Toolbar

- Untuk menampilkan tool yang ada didalam menu Flyout, klik menu flyout tersebut

24. Carilah fungsi menu file, menu slide show, menu insert, menu home, menu design, menu review and view, menu animation


Jawaban:

Menu-file: Menu file biasanya berisi fungsi-fungsi terkait file seperti membuka, menyimpan, mencetak, dan menutup presentasi atau dokumen yang sedang dikerjakan. Selain itu, menu file juga bisa digunakan untuk mengatur preferensi atau pengaturan aplikasi atau program yang digunakan.Menu-slide show: Menu slide show berisi fungsi-fungsi terkait tampilan presentasi, seperti memulai tampilan slide secara otomatis, mengatur durasi tampilan slide, menambahkan efek tampilan, dan lain sebagainya.Menu-insert: Menu insert berisi fungsi-fungsi terkait penambahan elemen baru ke dalam dokumen atau presentasi. Contohnya seperti menambahkan gambar, tabel, grafik, video, hyperlink, dan lain sebagainya.Menu-home: Menu home biasanya berisi fungsi-fungsi yang sering digunakan saat membuat dokumen atau presentasi, seperti pengaturan format font, paragraf, warna, alinea, dan lain sebagainya.Menu-design: Menu design berisi fungsi-fungsi terkait pengaturan tampilan atau desain presentasi, seperti memilih tema atau template, mengubah latar belakang, menyesuaikan warna, dan lain sebagainya.Menu-review and view: Menu review and view biasanya berisi fungsi-fungsi terkait pengecekan dan revisi dokumen atau presentasi, seperti menambahkan komentar, memeriksa ejaan dan tata bahasa, serta melakukan revisi atau perbaikan.Menu-animation: Menu animation berisi fungsi-fungsi terkait penambahan efek atau animasi pada tampilan presentasi, seperti animasi masuk, animasi keluar, dan animasi pergerakan objek lainnya. Fungsi-fungsi ini membantu untuk memperkaya tampilan presentasi dan membuatnya lebih menarik bagi audiens.
