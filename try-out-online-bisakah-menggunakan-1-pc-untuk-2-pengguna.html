15. D. 6. Fita ikut serta sebuah try out online program Primagama Mencari Juara bersama Hadi, pada program tersebut setiap paket try out berisi 10 soal dengan nilai 10 untuk jawaban benar, nilai -2 untuk jawaban salah dan -1 untuk soal yang tidak dijawab. Pada paket pertama Fita mengerjakan semua soal dan benar 5 soal, sedangkan pada paket kedua jawaban salah 1 soal dan 3 tidak dijawab dan untuk paket ketiga dia masing-masing 3 soal yang dia jawab salah dan tidak menjawab. Nilai rata-rata Fita pada try out tersebut adalah​


Untuk menghitung nilai rata-rata Fita pada try out tersebut, kita perlu menjumlahkan total nilai yang diperoleh Fita dari setiap paket try out, lalu membaginya dengan total jumlah soal yang dikerjakan. Berikut perhitungannya:

Paket pertama:

Jumlah soal: 10

Jawaban benar: 5

Jawaban salah: 0

Soal tidak dijawab: 0

Total nilai: (5 x 10) + (0 x -2) + (0 x -1) = 50

Paket kedua:

Jumlah soal: 10

Jawaban benar: 0

Jawaban salah: 1

Soal tidak dijawab: 3

Total nilai: (0 x 10) + (1 x -2) + (3 x -1) = -5

Paket ketiga:

Jumlah soal: 10

Jawaban benar: 0

Jawaban salah: 3

Soal tidak dijawab: 7

Total nilai: (0 x 10) + (3 x -2) + (7 x -1) = -17

Total nilai Fita dari ketiga paket try out = 50 + (-5) + (-17) = 28

Total jumlah soal yang dikerjakan = 10 + 10 + 10 = 30

Rata-rata nilai Fita = Total nilai / Total jumlah soal yang dikerjakan = 28 / 30 = 0.9333 (diambil 4 angka dibelakang koma)

Jadi, nilai rata-rata Fita pada try out tersebut adalah 0.9333.

Pilihan yang sesuai adalah  ( D )  Pernyataan  ( 1 )  SAJA cukup untuk menjawab pertanyaan, dan pernyataan  ( 2 )  SAJA cukup.

Penjelasan:

Ditanya:

Berapa nilai x

Pernyataan:

(1) 7AB = 5BC = 5CA

(2) 6AB = 5BC = 4CA

Jawab:

Untuk pernyataan (1)

7AB = 5BC

BC = [tex]\frac{7}{5} AB[/tex]

7AB = 5CA

CA  = [tex]\frac{7}{5} AB[/tex]

Dengan menggunakan aturan cosinus.

BC² = AB² + AC² - 2AB.AC.cos x

cos x = [tex]\frac{AB^2+AC^2-BC^2}{2.AB.AC}[/tex]

        = [tex]\frac{AB^2+(\frac{7}{5}AB)^2-(\frac{7}{5}AC)^2 }{2.AB.\frac{7}{5}AB}[/tex]

        = [tex]\frac{AB^2+\frac{49}{25}AB^2-\frac{49}{25}AB^2 }{\frac{14}{5}AB^2}[/tex]

        = [tex]\frac{AB^2}{\frac{14}{5}AB^2}[/tex]

        = [tex]\frac{1}{\frac{14}{5} }[/tex]

        = [tex]\frac{5}{14}[/tex]

x = arc cos [tex]\frac{5}{14}[/tex]

Untuk pernyataan (2) caranya sama, dengan menggunakan aturan cosinus.

Dengan demikian

Pilihan yang sesuai adalah  ( D )  Pernyataan  ( 1 )  SAJA cukup untuk menjawab pertanyaan, dan pernyataan  ( 2 )  SAJA cukup.

Jawaban:

Pilihan yang sesuai adalah  ( D )  Pernyataan  ( 1 )  SAJA cukup untuk menjawab pertanyaan, dan pernyataan  ( 2 )  SAJA cukup.

Penjelasan:

Diketahui:

AC = 7 cm

ABC siku-siku di C

CD merupakan garis tinggi

Ditanya:

Panjang CD

Jika tambahan diketahui: (1) BC = 5 dan (2) BD = 3

Jawab:

Untuk pernyataan (1) BC = 5

Panjang AB = [tex]\sqrt{AC^2+BC^2}[/tex]

                    = [tex]\sqrt{7^2+5^2}[/tex]

                    = [tex]\sqrt{49+25}[/tex]

                    = [tex]\sqrt{74}[/tex]

Panjang CD = [tex]\frac{AC*BC}{AB}[/tex]

                    = [tex]\frac{7*5}{\sqrt{74} }[/tex]

Untuk pernyataan (2) BD = 3

AD² + CD² = AC²    ⇒  AD² + CD² = 49

BD² + CD² = BC²    ⇒     9   + CD² = BC²

                                    ______________ -

                                    AD² - 9 = 49 - BC²

                                         BC² = 49 + 9 - AD²

                                         BC² = 58 - AD²

Pada segitiga ABC berlaku

A C ²  +  B C ²  =  A B ²

7 ²  + 58 - AD²   = ( A D  + 3)²

107 - AD²   = AD² + 6AD + 9

2AD² + 6AD - 98 = 0

Faktorkan, maka akan ketemu panjang AD

Jika Panjang AD sudah diketahui, Maka panjang CD juga dapat kita cari dengan pythagoras.
