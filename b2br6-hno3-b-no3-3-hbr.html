Cara perubahan biloks

Langkah 1 : Tentukan bilangan oksidasi masing masing unsur.

                      Reduksi

   Ι----------------------------------------------Ι

+1 +5 -2     +2 +5 -2     +1 +5 -2     +1 -1    +3 +5 -2

                  Ι-----------------------------------------Ι

                                Oksidasi

Langkah 2 : Setarakan atom atom yang berubah bilangan oksidasinya.

BrO₃⁻  +  Fe²⁺  ⇒  Br⁻ + Fe³⁺

Langkah 3 : Tentukan jumlah kenaikan dan penurunan bilangan oksidasi dengan mengalikannya dengan indeks.

                 Turun 6

   Ι---------------------------------Ι

BrO₃⁻  +  Fe²⁺  ⇒  Br⁻ + Fe³⁺

+5 -2        +2          -1       +3

                  Ι----------------------------Ι

                             Naik 1

Langkah 4 : Setarakan Jumlah kenaikan dan jumlah penurunan.

BrO₃⁻  +  6Fe²⁺  ⇒  Br⁻ + 6Fe³⁺

21. Bentuk setara 3Cu + HNO3 --> 3Cu(NO3)3 + NO + H2O adalah


Bentuk setara 3 Cu + HNO₃ --> 3 Cu(NO₃)₃ + NO + H₂O adalah

Jawab:

Jika kita ingin melakukan penyetaraan reaksi, maka hilangkan semua koefisien yang ada terlebih dahulu biar tidak membingunkan pada proses penyetaraan reaksinya,

Reaksi awal:

Cu + HNO₃ ==> Cu(NO₃)₃ + NO + H₂O

Penyetaraan reaksi redoks paling mudah menggunakan metode setengah reaksi, silahkan perhatikan langkah-langkahnya:

Sebelum mulai mereaksikan, maka kita hitung biloks masing-masing unsur terlebih dahulu untuk mengetahui unsur yang mengalami reduksi dan oksidasi.

Biloks Cu = 0

Biloks HNO₃ ==>

Biloks H + Biloks N + 3 x Biloks O = 0

+1 + Biloks N + 3 x (-2) = 0

Biloks N + (-5) = 0

Biloks N = +5

Biloks Cu(NO₃)₃ ==> Cu⁺³ + NO₃⁻ sehingga:

Biloks Cu⁺³ = + 3

Biloks NO₃⁻ = -1

Biloks N + 3 x Biloks O = -1

Biloks N + 3 x (-2) = -1

Biloks N = +5

Biloks NO ==>

Biloks N + Biloks O = 0

Biloks N + (-2) = 0

Biloks N = +2

Biloks H₂O ==>

Biloks H = +1

Biloks O = -2

Setelah semua biloks ditemukan, maka kita masuk ke langkah 1, yaitu:

Langkah 1: menuliskan kerangka dasar setengah reaksi reduksi dan setengah reaksi oksidasinya

Kita tuliskan reaksi ionnya saja, kesampingkan atom Hidrogen yang terikan.

Atom Hidrogen yang terikat sebagai kation menunjukkan reaksi berjalan dalam suasana asam.

Reaksi Oksidasi:

Cu ==> Cu⁺³

Reaksi Reduksi:

NO₃⁻ ==> NO

Langkah 2: Menyetarakan masing-masing setengah reaksi

a. Menyetarakan atom unsur yang mengalami perubahan Bilangan Oksidasi

Reaksi Oksidasi:

Cu ==> Cu⁺³  (Atom Cu sudah setara)

Reaksi Reduksi:

NO₃⁻ ==> NO (Atom N sudah setara)

b. Menyetarakan atom Oksigen dan Hidrogen (Oksigen terlebih dahulu)

Reaksi Oksidasi:

Cu ==> Cu⁺³

Karena tidak ada spesi lain maka tidak ada perlakuan pada bagian reaksi oksidasi.

Reaksi Reduksi:

NO₃⁻ ==> NO

karena reaksi berlangsung pada suasana asam, maka kita tambahkan molekul H₂O pada sisi yang kekurangan atom oksigen sebanyak selisih Oksigen yang ada, yaitu:

NO₃⁻ ==> NO + 2 H₂O

Selanjutnya kita setarakan atom Hidrogen dengan penambahan ion H⁺ (suasana asam) pada sisi satunya (kiri) sejumlah selisih atom Hidrogen yang ada, menjadi:

NO₃⁻ + 4 H⁺ ==> NO + 2 H₂O (selesai)

c. Menyetarakan Muatan

Penyetaraan muatan dilakukan dengan penambahan elektron disisi yang membutuhkan, yaitu:

Reaksi Oksidasi:

Cu ==> Cu⁺³

Muatan di sisi kiri = 0

Muatan di sisi kanan = +3

maka kita tambahkan 3 elektron di kanan, menjadi:

Cu ==> Cu⁺³ + 3 e⁻ (selesai)

Reaksi Reduksi:

NO₃⁻ + 4 H⁺ ==> NO + 2 H₂O

Muatan di kiri = -1 + 4 = +3

Muatan di kanan = 0

maka kita tambahkan 3 elektron di kiri, menjadi:

NO₃⁻ + 4 H⁺ + 3 e⁻ ==> NO + 2 H₂O  (selesai)

Langkah 3: Menyetarakan jumlah elektron

Menyetarakan jumlah elektron dengan menyamakan koefisien elektron dari reaksi oksidasi dan reduksi, karena pada reduksi koefisien elektron adalah 3 dan pada oksidasi adalah 3, maka kedua reaksi sudah memiliki jumlah elektron yang sama sehingga kita lanjutkan dengan melakukan pencoretan pada spesi yang sama pada sisi kanan dan kiri, yaitu:

Cu ==> Cu⁺³ + 3 e⁻

NO₃⁻ + 4 H⁺ + 3 e⁻ ==> NO + 2 H₂O

---------------------------------------------------------------  +

Cu + NO₃⁻ + 4 H⁺ ==> Cu⁺³ +  NO + 2 H₂O (setara)

Selanjutnya kita sederhanakan dengan menggabungkan ion H⁺ dan NO₃⁻ menjadi HNO₃, yaitu:

Cu + HNO₃ + 3 H⁺ ==> Cu⁺³ +  NO + 2 H₂O (setara)

Sisanya kita masukkan kembali NO₃⁻ sebanyak 3 molekul, menjadi:

Cu + HNO₃ + 3 HNO₃ ==> Cu(NO₃)₃ +  NO + 2 H₂O (setara)

Karena pada sisi kiri terdapat senyawa yang sama yaitu HNO₃, maka kita jadikan satu, yaitu:

Cu + 4 HNO₃ ==> Cu(NO₃)₃ +  NO + 2 H₂O (setara)

Selesai.

Pembahasan

Reaksi Redoks

Reaksi redoks adalah reaksi dimana terjadi perubahan biloks dari atom unsur sebelum dan sesudah reaksi, perubahan biloks tersebut berupa reaksi oksidasi yaitu kenaikan biloks dan reaksi reduksi yaitu penurunan biloks.

Oksidasi adalah pertambahan (kenaikan) bilangan oksidasi, sedangkan zat yang mengalami oksidasi disebut reduktor

Reduksi adalah penurunan bilangan bilangan oksidasi, sedangkan zat yang mengalami reduksi disebut oksidator

Aturan Penentuan Bilangan Oksidasi

Aturan Umum

1. Bilangan okidasi atom dalam unsur bebas sama dengan 0

2. Bilangan oksidasi ion monoatom sama dengan muatan ionnya

3. Jumlah biloks dalam Senyawa sama dengan 0. Sedangkan Jumlah biloks dalam Ion Poliatom sama dengan Muatan Ionnya.

4. Biloks unsur-unsur golongan VII A dalam senyawa biner logam adalah -1

5. Biloks H jika berikatan dg non logam sama dengan +1. Sedangkan biloks H jika barikatan dg logam dan boron adalah -1.

6. Biloks O dalam senyawa sama dengan -2, kecuali dalam senyawa biner fluorida, peroksida, dan superoksida.

7. Biloks logam golongan IA dalam senyawa sama dengan +1

8. Biloks logam golongan IIA dalam senyawa sama dengan +2


Penyetaraan reaksi redoks sederhana dapat disetarakan secara langsung tetapi jika persamaan reaksi redoks rumit/sulit akan sukar jika disetarakan secara langsung. Oleh karena itu, untuk persamaan reaksi redoks rumit dapat menggunakan metode bilangan oksidasi dan metode setengah reaksi untuk menyetarakan persamaan.


Jawab :

Penyetaran reaksi dari :

HNO₃ + FeSO₄ ---> Fe₂(SO₄)₃  + Fe(NO₃)₃ + NO + H₂O

Menentukan biloks masing-masing atom

HNO₃      +       FeSO₄     --->    Fe₂(SO₄)₃  +     Fe(NO₃)₃    + NO     + H₂O

+1   -2.3                 -2                        -2.3                  -1.3              -2       +1 -2

+1 +5 -6             +2 -2                +6   -6              +3     -3         +2  -2

                                                  +3

Atom Fe mengalami  oksidasi (kenaikan biloks) dari +2 menjadi +3

Atom N mengalami reduksi (penuruan biloks) dari +5 menjadi +2

Menyetarakan atom yang mengalami perubahan biloks

HNO₃      +       2FeSO₄     --->    Fe₂(SO₄)₃  +     Fe(NO₃)₃    + NO     + H₂O

Menentukan besarnya perubahan biloks pada masing-masing atom setelah atom disetarakan

HNO₃      +       2FeSO₄     --->    Fe₂(SO₄)₃  +     Fe(NO₃)₃    + NO     + H₂O

  +5                    +4                       +6                                         +2

Menetukan besarnya kenaikan dan penurunan biloks

HNO₃      +       2FeSO₄     --->    Fe₂(SO₄)₃  +     Fe(NO₃)₃    + NO     + H₂O

  +5                    +4                       +6                                         +2

   I___________I____________I_____-3_______________I

                            I____+2______I

Menyetarakan jumlah kenaikan dan penurunan biloks

HNO₃      +       2FeSO₄     --->    Fe₂(SO₄)₃  +     Fe(NO₃)₃    + NO     + H₂O

  +5                    +4                       +6                                         +2

   I___________I____________I_____-3 x 2___________I

                            I____+2 x 3___I

Menentukan persamaan reaksi setelah penyetaraan jumlah biloks

Atom Fe yang mengalami perubahan koefisien dikalikan 3

Atom N yang mengalami perubahan koefisien dikalikan 2

2HNO₃      +   6FeSO₄     --->    3Fe₂(SO₄)₃  +     Fe(NO₃)₃    +  2NO     + H₂O

Menyetarakan Kation, Anion, Hidrogen, Oksigen (KAHO)

2HNO₃      +   6FeSO₄     --->    3Fe₂(SO₄)₃  +     Fe(NO₃)₃    +  2NO     + H₂O

Kation = Fe (besi)

2HNO₃      +   9FeSO₄     --->    3Fe₂(SO₄)₃  +   3Fe(NO₃)₃    +  2NO     + H₂O

Anion = N (nitrogen)

10HNO₃      +   9FeSO₄     --->    3Fe₂(SO₄)₃  +   3Fe(NO₃)₃    +  NO     + H₂O

Hidrogen = H

12HNO₃  + 9FeSO₄  ---> 3Fe₂(SO₄)₃  +  3Fe(NO₃)₃  +  3NO  +  6H₂O

Oksigen = O

12HNO₃  + 9FeSO₄  ---> 3Fe₂(SO₄)₃  +  3Fe(NO₃)₃  +  3NO  +  6H₂O


Kesimpulan :

Reaksi setara adalah 12HNO₃  + 9FeSO₄  ---> 3Fe₂(SO₄)₃  +  3Fe(NO₃)₃  +  3NO  +  6H₂O

Al(s) dan HNO3(aq)b. Al(s) dan Al(NO3)2(aq)c. HNO3(aq) dan H,(g)d. Al(NO3)2(aq) dan H2(g)e. HNO3(aq) dan Al(NO3)2(aq)diuraikan ya kk tolong,,​

Al(s)+HNO3(aq)=>Al(NO3)3(aq)+H2(g)
